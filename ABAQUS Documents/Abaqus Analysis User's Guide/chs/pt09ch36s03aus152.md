# 36.3.8 扩展主面和滑移线

**产品：**Abaqus/Standard

##### **参考**

- ["在 Abaqus/Standard 中定义接触对," 第 36.3.1 节](pt09ch36s03aus145.md)
- ["Abaqus/Standard 接触建模相关的常见困难," 第 39.1.2 节](pt09ch39s01aus184.md)
- [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair)
- [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline)

### 概述

扩展主面或滑移线：

- 可以防止从节点在有限滑动问题中从主面（或滑移线）上“滑落”或卡在主面后方；
- 在小滑动和无限小滑动问题中，如果分析开始时从节点与主面没有交点，可以帮助从节点找到主面；
- 可以避免与接触建模相关的数值舍入困难；
- 不应替代正确的接触建模技术；
- 不应用于减少接触面底层单元数量；
- 在三维中只适用于主面的周边，在二维中只适用于主面的端部；并且
- 只适用于使用节点到面离散化的接触对。

### 为小滑动、节点到面接触扩展主面

如果从节点在分析开始时无法找到与主面的交点，它将可以自由穿透主面，因为不会形成局部切平面。这类问题通常发生在节点到面接触中，当从节点与主面端部或周边对齐（而主面没有绕过矩形体拐角）时，如 [图 36.3.8--1](pt09ch36s03aus152.md#aextsurf-small-slide) 所示；当使用前处理器生成节点坐标时，数值舍入误差也可能导致该问题。面内部的主面面片不会被扩展。如果 [图 36.3.8--1](pt09ch36s03aus152.md#aextsurf-small-slide) 中的主面被定义为绕过实体拐角，则不需要扩展主面，因为从节点会使用 ["Abaqus/Standard 中的接触公式," 第 38.1.1 节](pt09ch38s01aus177.md#usb-cni-acontactpairform-smsliding) 中讨论的投影方法投影到主面上。对于小滑动、面到面公式，图中所示情形通常不是问题，因为约束公式会考虑从节点附近的从面区域。

**图 36.3.8--1** 如果 *e*=*0*，在小滑动、节点到面接触中，从节点无法找到与主面的交点。

![](../graphics/aextsurf-small-slide-nls.png)

对于节点到面接触，可以将扩展区尺寸 *e* 指定为端部线段或面片边长的分数（见 [图 36.3.8--2](pt09ch36s03aus152.md#aextsurf-finite-slide)）。如果 *e* 设为零，Abaqus 将不扩展端部。给定值必须位于 0.0 和 0.2 之间。节点到面接触的默认值为 0.1；面到面接触不提供表面扩展。

| **输入文件用法：** | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), SMALL SLIDING, EXTENSION ZONE=*e* ``` |

### 在有限滑动、节点到面接触中扩展主面或滑移线

为了防止从节点从主面“滑落”或卡在主面后方，在有限滑动、节点到面接触中，可以将开放表面或滑移线扩展到其周边边（在三维中）或端节点（在二维中）之外。

可以将扩展区尺寸 *e* 指定为端部线段或面片边长的分数（见 [图 36.3.8--2](pt09ch36s03aus152.md#aextsurf-finite-slide)）。扩展区中的几何形状从端部线段或面片边外推得到。如果 *e* 设为零，Abaqus/Standard 将不扩展端部。给定值必须位于 0.0 和 0.2 之间。节点到面接触的默认值为 0.1。面到面接触不提供表面扩展；对于有限滑动、面到面接触，约束位于从面内，直到几乎整个从面面片滑离主面时才会发生“滑落”。只有在其他防止“滑落”的建模技术不可行，并且预计从节点只会在求解阶段的短时间内或非收敛迭代期间进入扩展区时，才应考虑用于有限滑动、节点到面接触的扩展。

| **输入文件用法：** | 使用以下任一选项： |
| --- | --- |
|  | ``` [*CONTACT PAIR](../key/key-link.md#usb-kws-hcontactpair), EXTENSION ZONE=*e* [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline), ELSET=*element_set_name*, EXTENSION ZONE=*e* ``` |
