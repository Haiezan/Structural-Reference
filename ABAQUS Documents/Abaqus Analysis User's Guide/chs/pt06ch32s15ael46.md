### 概述

本节提供 Abaqus/Standard 和 Abaqus/Explicit 中可用用户定义单元的参考信息。

##### 有效自由度

按用户单元定义中的规定。

##### 附加解变量

可以定义与未连接到其他单元的节点相关的解变量。不过，在 Abaqus/Standard 中，应避免在用户单元中使用 Lagrange 乘子定义约束，因为这可能导致方程求解器问题。

在 Abaqus/Explicit 中，不能使用 Lagrange 乘子定义约束，因为稳定时间增量会减小到无穷小。

### 所需节点坐标

线性用户单元不需要节点坐标。

对于一般用户单元，根据用户子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel)、[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 或 [`VUEL`](../sub/sub-link.md#sub-xsl-vuel) 的需要提供节点坐标。每个节点的最大坐标数在用户单元定义中指定（见 ["用户定义单元," 第 32.15.1 节](pt06ch32s15alm60.md#usb-elm-euserelem-coords) 中的 "Defining the maximum number of coordinates needed at any nodal point"）。每个节点的前几个坐标项应符合标准 Abaqus 约定（*X*, *Y*, *Z*，或轴对称单元的 *r*, *z*）。

### 单元属性定义

对于线性用户单元，属性为刚度和质量，这些属性通过用户定义矩阵定义，或从 Abaqus/Standard 结果文件读取。如有必要，可以在线性用户单元的单元属性定义中指定 Rayleigh 阻尼值。

对于通过用户子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel)、[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 或 [`VUEL`](../sub/sub-link.md#sub-xsl-vuel) 定义的一般用户单元，需要在用户单元定义中定义单元属性数量，并在单元属性定义中提供数值。这些属性的定义取决于你在子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel)、[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 或 [`VUEL`](../sub/sub-link.md#sub-xsl-vuel) 中的编码。

### 基于单元的载荷

线性用户单元无基于单元的载荷。

U*n*：对于一般用户单元，表示大小通过分布载荷或分布通量载荷定义给出的分布载荷或通量（见 ["分布载荷," 第 34.4.3 节](pt07ch34s04aus122.md) 或 ["热载荷," 第 34.4.4 节](pt07ch34s04aus123.md)）。*n* 必须为正整数，并会传入用户子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel)、[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 或 [`VUEL`](../sub/sub-link.md#sub-xsl-vuel)，以标识具体载荷类型。

U*n*NU：仅在 Abaqus/Standard 中可用。对于一般用户单元，表示在用户子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel) 或 [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 内部完全定义为等效节点值的分布载荷或通量。*n* 必须为正整数；当该载荷激活时，![](../graphics/usb_eqn05910.gif) 将传入子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel) 或 [`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat)，以标识载荷类型。*n* 前的负号表示该载荷为 NU 类型。

### 单元输出

对于线性用户单元，没有应力或应变分量，因为该单元只表现为刚度和质量。

对于一般用户单元，单元内的任何应力、应变或其他依赖解的变量都必须通过你在子程序 [`UEL`](../sub/sub-link.md#sub-xsl-uel)、[`UELMAT`](../sub/sub-link.md#sub-xsl-uelmat) 或 [`VUEL`](../sub/sub-link.md#sub-xsl-vuel) 中的编码定义为依赖解的状态变量。在 Abaqus/Standard 中，可以使用输出变量 SDV 输出它们。

当前不支持将用户定义单元的单元输出写入输出数据库。

### 单元上的节点顺序

按用户单元定义中的规定。
