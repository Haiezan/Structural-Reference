### 所需节点坐标

通用单元：无。

平面：*X*, *Y*

三维：*X*, *Y*, *Z*

轴对称：*r*, *z*

### 单元属性定义

对于通用单元，必须定义单元的表面积以及指向声学流体的声学流体-结构界面法向方向余弦。

对于用于平面模型的单元，必须指定单元的厚度（平面外）。如果未指定厚度，默认值为单位厚度。

对于用于三维和轴对称模型的单元，不需要附加数据。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Other** 作为截面 **Category**，选择 **Acoustic interface** 作为截面 **Type** |
| --- | --- |
|  | Abaqus/CAE 不支持通用声学界面截面。 |
