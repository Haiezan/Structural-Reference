### 分布式热通量

分布式热通量可用于具有温度自由度的单元。它们的指定如["Thermal loads," Section 34.4.4](pt07ch34s04aus123.md)中所述。

**负载 ID (*DFLUX)：**  BF**Abaqus/CAE 负载/交互：**  **物体热通量****单位：**[JL3 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **每单位体积的热体通量。

**负载 ID (*DFLUX)：**  BFNU**Abaqus/CAE 负载/交互：**  **物体热通量****单位：**[JL3 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **每单位体积的不均匀热体通量，其大小通过用户子程序[`DFLUX`](../sub/sub-link.md#sub-xsl-dflux)提供。

**负载 ID (*DFLUX)：**  S1**Abaqus/CAE 负载/交互：**  **表面热通量****单位：**[JL2 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **进入链路第一端（节点 1）的每单位面积的热表面通量。

**负载 ID (*DFLUX)：**  S2**Abaqus/CAE 负载/交互：**  **表面热通量****单位：**[JL2 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **进入链路第二端（节点 2 或节点 3）的每单位面积的热表面通量。

**负载 ID (*DFLUX)：**  S1NU**Abaqus/CAE 负载/交互：**  不支持**单位：**[JL2 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **进入链路第一端（节点 1）的每单位面积的非均匀热表面通量，其大小通过用户子程序[`DFLUX`](../sub/sub-link.md#sub-xsl-dflux)提供。

**负载 ID (*DFLUX)：**  S2NU**Abaqus/CAE 负载/交互：**  不支持**单位：**[JL2 T1](../popups/usb-int-iconventions-unitsym.md)**描述：  **进入链路第二端（节点 2 或节点 3）的每单位面积的非均匀热表面通量，其大小通过用户子程序[`DFLUX`](../sub/sub-link.md#sub-xsl-dflux)提供。
