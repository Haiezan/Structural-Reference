### 概述

Eulerian 单元：

- 只能用于显式动力分析；
- 必须具有八个唯一节点；
- 默认填充为空材料；
- 可以用非空材料初始化；
- 可以同时包含多种材料；并且
- 可以被材料部分填充。

### 典型应用

Eulerian 单元适用于涉及极端变形材料的仿真，包括流体流动。Eulerian 公式允许材料从一个单元流入另一个单元，同时 Eulerian 网格保持固定。使用 Eulerian 单元的应用见 Abaqus Benchmarks Guide 的 ["坍塌水柱的 Eulerian 分析," 第 1.7.1 节](../bmk/bmk-link.md#bmk-anl-watercollapse) 和 Abaqus Example Problems Guide 的 ["铆钉成形," 第 2.3.1 节](../exa/exa-link.md#exa-dyn-rivet)。

关于 Eulerian 分析的更多信息，见 ["Eulerian 分析," 第 14.1.1 节](pt04ch14s01aus90.md)。

### 选择合适的单元

可用的 Eulerian 单元为三维 8 节点单元 EC3D8R，以及三维 8 节点热耦合单元 EC3D8RT。二维仿真可以使用一层单元厚的网格或具有适当边界条件的楔形网格近似。Eulerian 网格通常是简单的矩形单元网格，并不贴合 Eulerian 材料的形状。复杂材料形状可以在该网格内部通过完全填充和部分填充单元以及周围空区域的组合来表示。

### 定义 Eulerian 单元的截面属性

必须将 Eulerian 截面定义与一组 Eulerian 单元关联。这组单元不得与其他类型单元共享节点。截面定义提供可能占据 Eulerian 单元的材料列表。

| **输入文件用法：** | ``` [*EULERIAN SECTION](../key/key-link.md#usb-kws-meulsection), ELSET=*element_set_name* *data lines giving list of materials* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Solid** 作为截面 **Category**，选择 **Eulerian** 作为截面 **Type**；**Assign Section**：选择部件 |
