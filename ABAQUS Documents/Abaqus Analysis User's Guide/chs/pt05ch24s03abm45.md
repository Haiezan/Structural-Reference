# 24.3.2 纤维增强复合材料的损伤起始

＃＃＃ 概述

纤维增强材料的材料损坏引发能力：
- 要求未损坏材料的行为是线性弹性的（参见["Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md)）；
- 基于 Hashin 理论（[Hashin and Rotem, 1973](#cdmagefibercomposite-hashin73)和[Hashin, 1980](#cdmagefibercomposite-hashin80)）；
- 考虑四种不同的失效模式：纤维拉伸、纤维压缩、基体拉伸和基体压缩；和
- 可与["Damage evolution and element removal for fiber-reinforced composites," Section 24.3.3](pt05ch24s03abm46.md)中描述的损伤演化模型结合使用（参见["Failure of blunt notched fiber metal laminates," Section 1.4.6 of the Abaqus Example Problems Guide](../exa/exa-link.md#exa-sta-damagefailfml)）。

### 伤害启动

### 元素

损坏引发准则必须与具有平面应力公式的单元一起使用，其中包括平面应力、壳、连续体壳和膜单元。

＃＃＃ 输出

除了 Abaqus 中可用的标准输出标识符（["Abaqus/Standard output variable identifiers," Section 4.2.1](pt02ch04s02abv01.md)和["Abaqus/Explicit output variable identifiers," Section 4.2.2](pt02ch04s02xbv01.md)）之外，以下变量还与纤维增强复合材料损伤模型中材料点处的损伤萌生特别相关：

| DMICRT |所有损坏引发标准组件。 |

| HSNFTCRT |分析过程中经历的纤维拉伸引发准则的最大值。 |
| --- | --- |

| HSNFFCCRT|分析过程中经历的纤维压缩引发准则的最大值。 |
| --- | --- |

| HSNMTCRT |分析期间经历的基体拉伸引发准则的最大值。 |
| --- | --- |

| HSNMCCRT |分析期间经历的基质压缩引发准则的最大值。 |
| --- | --- |

对于上述指示是否满足损坏模式中的启动准则的变量，小于1.0的值表示不满足准则，而1.0或更高的值表示满足准则。如果定义损伤演化模型，则该变量的最大值不超过 1.0。但是，如果您不定义损坏演化模型，则该变量的值可能高于 1.0，这表示超出标准的程度。
