### 元素属性定义

对于除广义平面应变单元之外的所有单元，您必须提供单元厚度；默认情况下，假定单位厚度。

对于广义平面应变单元，必须提供三个值：穿过参考节点的轴向材料纤维的初始长度、![](../graphics/usb_eqn04420.gif)的初始值（以弧度为单位）和![](../graphics/usb_eqn04421.gif)的初始值（以弧度为单位）。如果您不提供这些值，Abaqus 会假定![](../graphics/usb_eqn04420.gif)和![](../graphics/usb_eqn04421.gif)的初始长度为 1 个单位的默认值，并且为零。此外，您必须定义广义平面应变单元的参考点。

| **输入文件用法：** |使用以下选项定义除广义平面应变单元之外的所有单元的单元属性：
| --- | --- |
|  |``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection) ```使用以下选项定义广义平面应变单元的单元属性：``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), REF NODE=*node number or node set name* ```|

| **Abaqus/CAE 用法：** |属性模块： **创建截面**：选择 **实体** 作为截面 **类别** 和 **均匀**、**广义平面应变**，或 **电磁、固体** 作为截面 **类型** |
| --- | --- |
|  |广义平面应变截面必须分配给具有与其关联的参考点的零件区域。定义参考点： 零件模块：****工具**![](../graphics/images/arrow.gif)**参考点****：选择参考点 |
