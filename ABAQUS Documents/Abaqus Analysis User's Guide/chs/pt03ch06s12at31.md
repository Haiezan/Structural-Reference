# 6.12.1 退火过程

**产品：**Abaqus/Explicit  Abaqus/CAE

##### **参考**

- [*ANNEAL](../key/key-link.md#usb-kws-hanneal)
- Abaqus/CAE User's Guide 的 ["配置退火过程," 第 14.11.1 节](../usi/usi-link.md#usi-sim-configure-anneal)

### 概述

退火过程：

- 通过将所有适当的状态变量和速度设为零来对结构退火；并且
- 仅用于金属塑性和用户定义材料模型；对其他材料模型没有影响。

### 退火过程

退火过程用于模拟金属加热到高温时发生的应力和塑性应变松弛。从物理意义上讲，退火是将金属零件加热到高温，使微观组织重新结晶并消除材料冷加工所产生位错的过程。在退火过程中，Abaqus/Explicit 将所有适当的状态变量设为零。这些变量包括应力、背应力、塑性应变和速度。对于金属多孔塑性，空隙体积分数也设为零，使材料变为完全致密。

退火步没有时间尺度；因此，时间不会推进。退火过程瞬时发生。退火过程不需要数据。

#### 温度

热应变被设为零，并且模型中所有节点的温度将在退火过程中设为统一温度，或保持为当前温度。默认情况下，所有节点的温度保持当前温度。可以指定不同的最终温度 ![](../graphics/usb_eqn00123.gif)。

| **输入文件用法：** | ``` [*ANNEAL](../key/key-link.md#usb-kws-hanneal), TEMPERATURE=![](../graphics/usb_eqn00123.gif) ``` |

### 初始条件

退火步的初始状态为上一个显式动力分析步结束时模型的状态。

### 边界条件

在退火过程中不适合指定新的边界条件或修改边界条件；该过程之前生效的所有边界条件将保持固定。

### 载荷

在退火过程中指定载荷没有意义。

### 预定义场

在退火过程中指定预定义场没有意义。

### 材料选项

退火过程仅用于金属塑性模型（["经典金属塑性," 第 23.2.1 节](pt05ch23s02abm17.md)）以及使用用户子程序 [`VFABRIC`](../sub/sub-link.md#sub-xsl-vfabric) 和 [`VUMAT`](../sub/sub-link.md#sub-xsl-vumat) 建模的用户定义材料。Abaqus/Explicit 中的金属塑性模型包括 Mises、Johnson-Cook、Hill 和金属多孔塑性。Abaqus/Explicit 还允许对弹性材料（["线弹性行为," 第 22.2.1 节](pt05ch22s02abm02.md)）进行退火，包括各向同性、正交各向异性和各向异性弹性。退火过程对其他材料模型没有影响。

### 单元

Abaqus/Explicit 中可用的所有单元都可以用于退火过程。这些单元列在 [第 VI 部分，“单元”](pt06.md) 中。

### 输出

退火步没有相关输出。
