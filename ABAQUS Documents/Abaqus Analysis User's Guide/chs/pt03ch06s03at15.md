＃＃＃ 概述

响应谱分析：
- 提供结构对固定点动态运动（“基本运动”）或动态力的峰值线性响应的估计；
- 通常用于分析对地震事件的响应；
- 假设系统的响应是线性的，以便可以使用其自然模式在频域中进行分析，该自然模式必须在先前的特征频率提取步骤（["Natural frequency extraction," Section 6.3.5](pt03ch06s03at10.md)）中提取；
- 可使用高性能SIM软件架构（参见["Using the SIM architecture for modal superposition dynamic analyses" in "Dynamic analysis procedures: overview," Section 6.3.1](pt03ch06s03abo07.md#usb-anl-alineardynamics)）；和
- 是一个线性扰动过程，因此，如果激励非常严重以至于系统中的非线性效应很重要，则不合适。

### 响应谱分析

响应谱分析可用于估计结构对特定基础运动或力的峰值响应（位移、应力等）。该方法只是近似方法，但对于初步设计研究来说，它通常是一种有用且廉价的方法。

响应谱过程基于使用系统模式的子集，必须首先使用特征频率提取过程来提取该子集。这些模式将包括本征模式，如果在本征频率提取步骤中激活，则还包括残余模式。提取的模式数量必须足以对系统的动态响应进行充分建模，这取决于您的判断。

在具有重复特征值和特征向量的情况下，必须谨慎解释模态求和结果。您应该向结构添加微不足道的质量或扰乱对称几何形状，以使特征值变得唯一。

虽然响应谱过程中的响应是线性振动，但先前的响应可能是非线性的。如果在特征频率提取步骤之前的一般分析步骤中包含非线性几何效应 (["General and linear perturbation procedures," Section 6.1.3](pt03ch06s01aus44.md))，则初始应力效应（应力刚化）将包含在响应谱分析中。

要解决的问题可以表述如下：给定一组基本运动![](../graphics/usb_eqn00697.gif)(![](../graphics/usb_eqn00698.gif))，在由方向余弦![](../graphics/usb_eqn00699.gif)(![](../graphics/usb_eqn00700.gif)) 定义的正交方向上指定，估计同时受到这些多个基本运动的有限元模型中任何变量的响应的所有时间的峰值。首先针对系统​​的每个固有模式的每个激励方向独立计算峰值响应，作为频率和阻尼的函数。然后将这些独立响应组合起来，以创建对选择用于输出的任何变量的实际峰值响应的估计，作为频率和阻尼的函数。

加速度历史（基本运动）在响应谱分析中不直接给出；它必须首先转换成频谱。

### 指定频谱

响应谱方法基于首先找到固有频率等于感兴趣频率的单自由度系统的每个基本运动激励的峰值响应。单自由度系统的特征在于其无阻尼固有频率![](../graphics/usb_eqn00701.gif)以及每个模式![](../graphics/usb_eqn00156.gif)下系统中存在的临界阻尼分数![](../graphics/usb_eqn00564.gif)。系统的运动方程随时间积分，以找到线性单自由度系统的相对位移、相对速度以及相对或绝对加速度的峰值。对感兴趣范围内的所有频率和阻尼值重复此过程。这些响应的图称为位移、速度和加速度谱：![](../graphics/usb_eqn00702.gif)、![](../graphics/usb_eqn00703.gif)和![](../graphics/usb_eqn00704.gif)。响应谱可以直接从测量数据中获得，如下面的["Defining a spectrum using values of *S* as a function of frequency and damping](pt03ch06s03at15.md#usb-anl-aresponsespectrum-define)中所述。您还可以使用 FORTRAN 程序来定义谱；["Analysis of a cantilever subject to earthquake motion," Section 1.4.13 of the Abaqus Benchmarks Guide](../bmk/bmk-link.md#bmk-anl-cantilever)中提供了以这种方式从加速度记录定义谱的示例。

或者，您可以通过指定要构建频谱的幅度（时间历史记录）、频率范围和阻尼值来创建所需的频谱，如下面的["Creating a spectrum from a given time history record](pt03ch06s03at15.md#usb-anl-aresponsespectrum-create)中所述。该频谱可以用于后续的响应频谱分析，也可以写入文件以供将来使用。

对于每个阻尼值，必须在所需的整个频率范围内以频率的升序给出响应谱的幅度。 Abaqus/Standard 在双对数刻度上给出的值之间进行线性插值。在给定频率范围的极值之外，假定幅度是恒定的，对应于给定的最终值。 （有关数据插值的说明，请参阅["Material data definition," Section 21.1.2](pt05ch21s01aus109.md)。）

可以定义任意数量的光谱，并且每个光谱都必须命名。响应谱过程允许最多三个谱在由其方向余弦定义的正交物理方向上同时应用于模型。

### 结合各个峰值响应

对不同方向激励的各个峰值响应将在不同时间发生，因此必须组合成总体峰值响应。必须执行两种组合，并且都将近似值引入到结果中：

1. 多向激励必须组合成一个整体响应。这种组合由方向求和方法控制，如下["Directional summation methods](pt03ch06s03at15.md#usb-anl-aresponsespectrum-directsum)中所述。”
2. 必须结合峰值模态响应来估计峰值物理响应。这种组合由模态求和方法控制，如下["Modal summation methods](pt03ch06s03at15.md#usb-anl-aresponsespectrum-modalsum)中所述。”

根据基本激励的类型，首先组合模态响应或方向响应。

#### 选择模式和指定阻尼系数的规则

以下规则适用于选择模态和指定模态阻尼系数：
- 默认情况下不包含模态阻尼。
- 模式选择和模态阻尼必须以相同的方式指定，使用模式编号或频率范围。
- 如果您不选择任何模式，则先前频率分析中提取的所有模式（包括剩余模式（如果已激活））将用于叠加。
- 如果您没有为所选模式指定阻尼系数，则这些模式将使用零阻尼值。
- 阻尼仅应用于选定的模式。
- 超出指定频率范围的所选模式的阻尼系数是恒定的，并且等于为第一个或最后一个频率指定的阻尼系数（取决于哪个频率更接近）。这与 Abaqus 解释幅度定义的方式一致。

### 初始条件

在响应谱分析中指定初始条件是不合适的。

### 边界条件

假设受边界条件约束的所有点和连接器元件的接地节点在任一方向上同相移动。该基本运动可以在三个正交方向（二维模型中的两个方向）中的每个方向上使用不同的输入频谱。您可以将输入频谱![](../graphics/usb_eqn00745.gif)定义为频率![](../graphics/usb_eqn00348.gif)的函数，针对不同的临界阻尼值![](../graphics/usb_eqn00746.gif)，如前面在["Specifying a spectrum](pt03ch06s03at15.md#usb-anl-aresponsespectrum-spec)中所述。”二级碱基不能用于响应谱分析。

### 负载

如前所述，可以在响应谱分析中定义的唯一“载荷”是由输入谱定义的“载荷”。在响应谱分析中不能规定其他载荷。

### 预定义字段

预定义字段（包括温度）不能用于响应谱分析。

### 材料选项

必须定义材料的密度 (["Density," Section 21.2.1](pt05ch21s02abm01.md))。在响应谱分析期间，以下材料属性不处于活动状态：塑性和其他非弹性效应、速率相关的材料属性、热属性、质量扩散属性、电属性和孔隙流体流动属性 - 请参阅["General and linear perturbation procedures," Section 6.1.3](pt03ch06s01aus44.md)。

### 元素

除了带有扭曲的广义轴对称单元之外，Abaqus/Standard 中的任何应力/位移单元都可以用于响应谱分析 - 请参阅["Choosing the appropriate element for an analysis type," Section 27.1.3](pt06ch27s01aus112.md)。

使用基于 SIM 的频率提取程序以及 AMS 或 Lanczos 特征求解器提取的特征模态进行响应谱分析时，不支持反作用力输出。响应谱分析中的反作用力输出使用默认 Lanczos 本征解算器提取的本征模态，提供所谓的模态反作用力的方向组合，并以相应广义位移的最大绝对值加权。用于反作用力计算的方向和模态组合规则与其他节点输出变量相同。  模态反作用力在频率提取过程中计算。它们代表为简正振型计算的静态反作用力。一般来说，它们不能充分代表动态分析中的反作用力。对于具有对角质量和对角阻尼矩阵的模型，模态反作用力的叠加可以在基于模态的分析（而不是响应谱分析）中提供节点反作用力的合理近似。在响应谱分析中，模型响应可以通过包含支撑节点的结构元件中请求的截面应力和截面力来更好地表示。
