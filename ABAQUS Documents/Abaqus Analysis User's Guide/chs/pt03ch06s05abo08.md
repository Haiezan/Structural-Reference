# 6.5.1 传热分析程序：概述







Abaqus 可以解决以下类型的传热问题：

**非耦合传热分析**：涉及传导、强制对流和边界辐射的传热问题可以在 Abaqus/Standard 或 Abaqus/CFD 中进行分析。参见["Uncoupled heat transfer analysis," Section 6.5.2](pt03ch06s05at18.md)。在这些分析中，计算温度场时不了解所研究物体的应力/变形状态或电场。纯传热问题可以是瞬态的或稳态的，也可以是线性的或非线性的。

**顺序耦合热应力分析**：如果应力/位移解依赖于温度场，但不存在逆相关性，则可以在 Abaqus/Standard 中进行顺序耦合热应力分析。顺序耦合热应力分析是通过首先求解纯传热问题，然后将温度解读入应力分析作为预定义字段来执行的。参见["Sequentially coupled thermal-stress analysis," Section 16.1.2](pt04ch16s01at39.md)。在应力分析中，温度可能随时间和位置而变化，但应力分析解决方案不会改变。 Abaqus 允许传热分析模型和热应力分析模型之间使用不同的网格。温度值将根据在热应力模型节点处评估的元素插值器进行插值。

**完全耦合热应力分析**：耦合温度位移程序用于同时求解应力/位移和温度场。当热解和机械解相互影响强烈时，使用耦合分析。例如，在快速金属加工问题中，材料的非弹性变形会导致发热，而在接触问题中，跨间隙传导的热量可能很大程度上取决于间隙间隙或压力。Abaqus/Standard 和 Abaqus/Explicit 都提供耦合温度-位移分析程序，但每个程序使用的算法有很大不同。在 Abaqus/Standard 中，使用后向差分格式对传热方程进行积分，并使用牛顿法求解耦合系统。这些问题可以是瞬态的或稳态的，也可以是线性的或非线性的。在 Abaqus/Explicit 中，使用显式前向差分时间积分规则对传热方程进行积分，并使用显式中心差分积分规则获得力学解响应。 Abaqus/Explicit 中的完全耦合热应力分析始终是瞬态的。空腔辐射效应不能包含在全耦合热应力分析中。有关详细信息，请参阅["Fully coupled thermal-stress analysis," Section 6.5.3](pt03ch06s05at19.md)。

**完全耦合热电结构分析**：耦合热电结构程序用于同时求解应力/位移、电势和温度场。当热、电和机械解决方案相互强烈影响时，使用耦合分析。这种工艺的一个例子是电阻点焊，其中两个或多个金属部件通过在材料界面处的离散点处熔合而连接。熔合是由接触点处的电流产生的热量引起的，这取决于施加在这些点上的压力。这些问题可以是瞬态的或稳态的，也可以是线性的或非线性的。  空腔辐射效应不能包含在完全耦合的热电结构分析中。此过程仅在 Abaqus/Standard 中可用。有关详细信息，请参阅["Fully coupled thermal-electrical-structural analysis," Section 6.7.4](pt03ch06s07at23.md)。

**绝热分析**：绝热机械分析可用于机械变形导致加热的情况，但该事件速度太快，热量没有时间扩散到材料中。绝热分析可以在 Abaqus/Standard 或 Abaqus/Explicit 中进行；参见["Adiabatic analysis," Section 6.5.4](pt03ch06s05at20.md)。绝热分析可以是静态的或动态的、线性的或非线性的。

**耦合热电分析**：Abaqus/Standard 提供了完全耦合的热电分析功能，用于解决由于电流流过导体而产生热量的问题。参见["Coupled thermal-electrical analysis," Section 6.7.3](pt03ch06s07at22.md)。

**空腔辐射**：在 Abaqus/Standard 中，空腔辐射效应（除了规定的边界辐射之外）还可以包含在非耦合传热问题中。参见["Cavity radiation," Section 41.1.1](pt09ch41s01aus187.md)。空腔可以是开放的或封闭的。可以对空腔内的对称性和阻塞进行建模。自动计算视角系数，并且可以在分析过程中指定包围空腔的物体的运动。空腔辐射问题是非线性的，可以是瞬态的，也可以是稳态的。
