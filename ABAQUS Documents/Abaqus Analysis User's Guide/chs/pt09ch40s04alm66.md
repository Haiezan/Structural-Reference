# 40.4.1 滑动线接触元素







**产品：**Abaqus/标准  

＃＃＃＃＃ **参考**

-["Axisymmetric slide line element library," Section 40.4.2](pt09ch40s04ael51.md)-[*INTERFACE](../key/key-link.md#usb-kws-minterface)-[*SLIDE LINE](../key/key-link.md#usb-kws-mslideline)

＃＃＃ 概述

滑动线元素：
- 当滑动沿特定平面内的线（“滑动线”）发生时，可以对两个变形体之间的有限滑动相互作用进行建模；
- 假设与滑动线正交的切向运动为零或很小（Abaqus/Standard 将此类运动视为无穷小）；
- 可与轴对称应力/位移元件一起使用；
- 建议用于特定应用，例如当接触表面是子结构的表面或当接触涉及 CAXA 或 SAXA 单元时；
- 适用于一阶和二阶元素；和
- 使用相同的“主从”概念来实施基于表面的接触中所见的接触约束。

有关接触建模的一般讨论，请参阅[Chapter 36, "Defining Contact Interactions](pt09ch36.md)。”

### 滑线和滑线元素的主从概念

创建包含滑移线元素的模型时，请记住 Abaqus/Standard 使用严格的“主从”概念来强制接触约束，这一点很有用。滑线接触元件形成“从”表面。您指定用于定义滑动线的节点定义“主”曲面。滑动线接触元件的节点被限制为不穿透主表面。

无论是使用曲面还是单元来定义接触，选择主曲面和从曲面的注意事项都是相同的。如果材料不同，主表面应选择刚体较硬的表面，或者选择网格较粗的表面。如果两个表面的材料和网格密度相同，则选择是任意的。

### 定义滑动线（主表面）

您可以指定构成滑动线的节点，也可以按如下所述生成它们。如果选择直接指定节点，则必须按定义连续滑动线的顺序指定它们。节点序列定义滑动线的切向量![](../graphics/usb_eqn01319.gif)。滑动线可以由线性或抛物线段组成，具体取决于模型是由一阶元素还是二阶元素组成。在任一情况下，可以通过平滑滑动线来改善收敛。

#### 定义线性滑动线

当实体的表面与一阶单元啮合时，定义由线性单元段组成的滑动线。如[Figure 40.4.1--4](pt09ch40s04alm66.md#eslideline-1st-order)所示），节点 *i*、*j*、*k*、...、*p* 按该顺序指定，从而标识从 *i* 到 *p* 的滑动线。节点 *I*、*J*、*K*、...、*N* 是与该滑动线关联的 ISL 类型元素的节点。

| **输入文件用法：** |``` [*SLIDE LINE](../key/key-link.md#usb-kws-mslideline), ELSET=*element_set_name*, TYPE=LINEAR *first node number, second node number, etc.* ```|
| --- | --- |

#### 定义抛物线滑动线

当实体的表面与二阶单元网格化时，定义由二阶单元段组成的滑动线。在这种情况下，滑动线应由奇数个节点组成。如[Figure 40.4.1--5](pt09ch40s04alm66.md#eslideline-2nd-order)所示，节点 *i*、*j*、*k*、...、*u* 按该顺序指定，从而标识从 *i* 到 *u* 的滑动线。节点 *I*、*J*、*K*、...、*O* 是与该滑动线关联的 ISL 类型元素的节点。

#### 平滑滑动线

通常通过平滑滑动线段之间的表面切线中的不连续性来改进会聚，从而提供沿滑动线平滑变化的切线。有关平滑滑动线的详细信息，请参见["Contact formulations in Abaqus/Standard," Section 38.1.1](pt09ch38s01aus177.md)。

### 定义滑动线元素（从属面）

许多有限滑动接触模拟可以使用[Chapter 36, "Defining Contact Interactions](pt09ch36.md)中描述的基于表面的接触方法来定义模型。仅针对特定应用，建议使用轴对称应力/位移和耦合温度位移滑线单元，例如当接触面是子结构的表面或接触涉及 CAXA 或 SAXA 单元时（请参阅["Contact modeling if asymmetric-axisymmetric elements are present," Section 36.3.10](pt09ch36s03aus154.md)）。

滑动线接触元件限定从属表面。使用滑动线接触单元的当前长度和分配给该单元的常量“宽度”来计算与从属表面上的每个节点关联的接触面积，该常量“宽度”取决于底层有限元。

### 定义与滑动线元素的非默认机械表面交互

默认情况下，Abaqus/Standard 使用与滑移线元件的“硬”、无摩擦接触。您可以指定可选的机械表面交互模型。可以使用以下机械表面相互作用模型：
- 摩擦。详细信息请参见["Frictional behavior," Section 37.1.5](pt09ch37s01aus169.md)。
- 修改了“硬”接触、软化接触和粘性阻尼。有关详细信息，请参阅["Contact pressure-overclosure relationships," Section 37.1.2](pt09ch37s01aus166.md)和["Contact damping," Section 37.1.3](pt09ch37s01aus167.md)。
