### 概述

声学界面单元：

- 可用于将声学流体模型与包含连续体或结构单元的结构模型耦合；
- 将结构模型表面的加速度与声学介质中的压力耦合；
- 可用于动力和稳态动力过程；
- 必须使用声学单元和结构（或实体）单元共享的节点定义；
- 只能用于小位移仿真，不适用于非线性或静水流固相互作用；
- 如果使用子空间迭代特征求解器，则在特征频率提取分析中会被忽略；并且
- 如有必要，可退化为三角形单元。

对于多数问题，["网格绑定约束," 第 35.3.1 节](pt08ch35s03aus132.md) 和 ["在 Abaqus/Standard 中定义绑定接触," 第 36.3.7 节](pt09ch36s03aus151.md) 中描述的基于面的结构-声学功能，为声学流体与结构之间的相互作用建模提供了更通用且更易用的方法。用户指定的声学界面单元可以提高对耦合定义的控制能力，但代价是失去基于面过程的便利性。

### 典型应用

声学界面单元用于固体结构运动会影响声学流体压力的仿真，例如汽车车架振动在乘员舱内产生噪声；也用于流体压力影响邻近结构的仿真，例如容器内小幅晃动流体影响容器响应。

用户指定的声学界面单元也适用于只包含声学介质的问题，因为它们允许直接在声学界面单元节点上指定位移、速度或加速度边界条件。不过，在这种应用中必须注意，切向位移不会与流体耦合。因此，如果这些节点在切向方向上未受约束，可能出现涉及位移自由度的零能模式。当声学界面单元用于耦合流体和实体单元时，由于实体的刚度和惯性，不会出现此问题。

### 选择合适的单元

底层声学单元和结构单元的阶次通常决定应使用哪种声学界面单元。通用声学界面单元 ASI1 可用于任何耦合声学-结构仿真；但通常只与声学 link 单元（AC1D2 和 AC1D3）一起使用。

### 定义声学界面单元的截面属性

必须将声学界面截面定义与一组声学界面单元关联。该截面定义必须与三维和轴对称声学界面单元一起使用，即使这些单元没有用户定义的几何属性。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface), ELSET=*element_set_name* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Other** 作为截面 **Category**，选择 **Acoustic interface** 作为截面 **Type**；**Assign Section**：选择区域 |
| --- | --- |

#### 定义与 ASI1 单元相关的几何属性

ASI1 单元由单个节点组成。Abaqus/Standard 无法计算与这些单元相关的表面积，因此必须提供该信息。如果未给出准确的表面积，Abaqus/Standard 可能会在声学-结构界面处计算出不正确的加速度或声学流体压力。

此外，Abaqus/Standard 无法计算与这些单元相关的界面法向。必须在全局笛卡尔坐标系中提供这些单元界面法向的方向余弦。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) *surface area*, *X**-direction cosine*, *Y**-direction cosine*, *Z**-direction cosine* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Abaqus/CAE 不支持通用声学界面截面。 |
| --- | --- |

#### 定义平面声学界面单元的厚度

可以指定平面声学界面单元的厚度。默认值为单位厚度。

| **输入文件用法：** | ``` [*INTERFACE](../key/key-link.md#usb-kws-minterface) *thickness* ``` |
| --- | --- |

| **Abaqus/CAE 用法：** | Property 模块：**Create Section**：选择 **Other** 作为截面 **Category**，选择 **Acoustic interface** 作为截面 **Type**：**Plane stress/strain thickness**：*thickness* |
| --- | --- |

### 在不同插值阶次的单元形成声学-结构界面时使用声学界面单元

通常假定声学流体网格和结构网格（至少在界面表面处）使用相同阶次的插值。如果不是这种情况，必须沿声学-结构界面对节点施加适当的 MPC，以保持压力场（MPC 类型 P LINEAR）或位移场（MPC 类型 LINEAR）的相容性。
