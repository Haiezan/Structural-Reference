### 概述

连续粒子单元：

- 只能用于显式动力分析；
- 必须只有一个节点；
- 具有一个积分点；
- 可以按类似连续体单元的方式初始化；并且
- 完全由材料填充。

### 典型应用

连续粒子单元（PC3D）适用于涉及极端变形材料的仿真，例如自由表面流体流动或固体结构的破碎/碎裂。它们只使用一个节点定义；但是，以给定节点（粒子）为中心的单元会接收影响球内所有粒子的贡献，该影响球的半径通常称为光滑长度。光滑粒子流体动力学（SPH）公式在分析的每个增量中确定与给定粒子相关的连接关系。由于节点连接关系不是固定的，因此避免了严重的单元畸变，从而使该公式能够处理很高的应变梯度。

1 节点 PC3D 单元用于定义待建模物体表面和内部的点。可以按类似质量单元的方式定义这些节点，并且这些节点可以像常规砖单元网格的节点一样布置在空间中。光滑粒子流体动力学网格通常是一个均匀间距的单元网格，并符合待建模物体的形状。

更多信息见 ["光滑粒子流体动力学," 第 15.2.1 节](pt04ch15s02aus95.md)。

### 定义单元的截面属性

必须将实体截面定义与一组连续粒子单元关联。截面定义提供与 PC3D 单元相关的材料。

作为实体截面定义的一部分，可以定义特征长度。该特征长度不要与光滑长度混淆，用于计算粒子体积。该体积假定为一个立方体，其边长等于指定特征长度的两倍。

| **输入文件用法：** | ``` [*SOLID SECTION](../key/key-link.md#usb-kws-msolidsection), ELSET=*element_set_name* *characteristic length associated with the particle volume* ``` |
| --- | --- |
|  | 其中 ELSET 参数引用一组粒子单元。 |
