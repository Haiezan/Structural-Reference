# 2.1.2 参数形状变化







**产品：**Abaqus/标准 Abaqus/Explicit  

＃＃＃＃＃ **参考**

-["Parametric input," Section 1.4.1](pt01ch01s04aus04.md)-[*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape)

＃＃＃ 概述

形状参数化可以通过以下方式在 Abaqus 输入文件中完成： 
- 参数化节点坐标；或
- 使用形状变化将节点坐标与形状参数相关联。

本节描述形状参数化的不同方法。

### 节点坐标的参数化

任何单独的节点坐标都可以直接参数化。这通常价值有限，因为它通常会导致设计形状不规则且不易制造。此外，单个节点坐标的参数化通常需要过多的参数来定义参数化形状。

与 Abaqus 中的节点生成结合使用的节点坐标参数化提供了一种更实用的形状参数化方法。然而，这种方法的实际用途仍然有限，因为 Abaqus 中可用的简单节点生成功能无法描述复杂的形状。

#### 单个节点坐标的直接参数化

节点坐标参数化的最简单形式是定义各个参数并使用它们代替要参数化的节点坐标，如["Parametric input," Section 1.4.1](pt01ch01s04aus04.md)中所述。例如，

```
[*PARAMETER](../key/key-link.md#usb-kws-mparameter)
x_coord_node_1 = 10。
y_coord_node_1 = 20。
[*NODE](../key/key-link.md#usb-kws-mnode)
1、<x_coord_node_1>、<y_coord_node_1>
```

#### 使用节点生成对节点坐标进行参数化

形状参数化可以通过参数化某些节点的坐标，然后使用这些节点生成其他节点及其坐标来完成。例如：

```
[*PARAMETER](../key/key-link.md#usb-kws-mparameter)
x_coord_node_1 = 10。
x_coord_node_11 = 20。
[*NODE](../key/key-link.md#usb-kws-mnode)
1、<x_coord_node_1>，50。
11、<x_coord_node_11>，50。
[*NGEN](../key/key-link.md#usb-kws-mngen)
1, 11, 1
```
这种形状参数化方法通过隐式地使生成节点的节点坐标依赖于形状参数，减少了形状参数化所需的用户定义参数的数量。

Abaqus 中形状的定义包括基本形状加上使用线性组合添加到基本形状的任意数量的附加形状变化。在数学上，我们可以将节点坐标![](../graphics/usb_eqn00143.gif)表示为

您可以通过直接或通过节点生成提供节点定义来指定 Abaqus 输入文件中模型的基本形状；参见["Node definition," Section 2.1.1](pt01ch02s01aus05.md)。

您可以指定形状变化和关联的形状参数，如此处所述。

此外，您可以将形状的扰动指定为其他形状（例如屈曲模式形状）的线性组合；参见["Introducing a geometric imperfection into a model," Section 11.3.1](pt04ch11s03aus67.md)。

然后可以使用四种方法的组合来定义 Abaqus 输入文件中模型的节点坐标： 
- 可以直接定义各个节点及其各自的坐标；这些坐标是基本形状![](../graphics/usb_eqn00145.gif)定义的一部分，并且可以参数化。
- 节点生成可用于根据依赖于现有节点定义的几何简单映射来创建节点及其坐标；这些生成的坐标也是基本形状![](../graphics/usb_eqn00145.gif)定义的一部分。如有必要，可以对节点生成输入进行参数化。
- 参数形状变化可用于改变使用上述方法定义的节点的坐标。
- 几何缺陷可用于扰乱先前使用上述三种方法的任意组合定义的节点坐标。

### 使用形状变化进行形状参数化

您可以指定形状变化，而不是直接参数化节点坐标。每个形状变化必须与单个形状参数相关联。必须选择与形状变化相关的参数名称，以便在以不区分大小写的方式解释时名称保持唯一。形状参数的值是使用参数定义来分配的。

可以为同一参数多次定义参数形状变化，以便可以单独指定形状变化的不同部分。在这些情况下，如果在多个参数形状变化定义中指定了同一节点，则以该节点的最后一个定义为准。

在参数形状变化定义下指定的、尚未直接定义或通过节点生成定义的节点将被忽略。

您可以使用三种可能性的组合来指定形状变化：直接指定它们、从备用输入文件中读取它们以及从辅助分析的结果文件中读取它们。以下各节介绍了这些方法。

#### 直接定义形状变化或从备用输入文件中读取它们

您可以通过指定节点号和相应的坐标分量变化来直接定义形状变化数据。或者，数据可以在 ASCII 文件中给出。

| **输入文件用法：** |使用以下选项直接指定形状变化数据： |
| --- | --- |
|  |``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name* ```使用以下选项指定备用输入文件中的形状变化数据：``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, INPUT=*input file* ```|

##### 在替代坐标系中定义形状变化

默认情况下，形状变化数据在全局直角笛卡尔坐标系中解释。您可以在圆柱或球坐标系中指定形状变化数据（直接或在备用输入文件中）。在这种情况下，形状变化的计算如下进行。定义基本形状的节点坐标分量首先从存储它们的全局直角笛卡尔坐标系转换到指定的坐标系。然后将形状变化坐标分量相加以给出更新的坐标分量，将其变换回全局直角笛卡尔坐标系。最后，使用全局直角笛卡尔坐标系中表达的分量，将形状变化视为更新的坐标分量与原始坐标分量之间的差。在形状变化的计算中的任何点都不使用与形状变化相关联的形状参数的值。

| **输入文件用法：** |使用以下选项指定直角坐标系（默认）中的形状变化数据： |
| --- | --- |
|  |``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, SYSTEM=R ```使用以下选项指定圆柱坐标系中的形状变化数据：``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, SYSTEM=C ```使用以下选项指定球面坐标系中的形状变化数据：``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, SYSTEM=S ```|``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, SYSTEM=R ```

#### 使用辅助分析来生成形状变化

辅助模型是附加的有限元模型，用于生成主模型的形状变化。可以使用辅助模型来简化此过程，而不是直接逐个节点地定义形状变化。辅助分析是这些辅助模型的有限元分析。

辅助模型通常具有与主模型相同的几何形状、单元连接性和材料类型。然而，边界条件通常是不同的。对辅助模型施加载荷会产生一组位移，我们可以将其解释为形状变化。例如，我们可能有兴趣研究结构的非线性屈曲行为相对于结构缺陷的敏感性。在这种情况下，我们可以执行辅助特征值线性屈曲分析，然后使用生成的振型作为要添加到主模型的基本几何形状的形状变化。 （这个特殊问题也可以通过使用几何缺陷来解决。）

Abaqus 通过用户节点标签从辅助分析中读取形状变化数据。 Abaqus 不检查两次分析运行之间的模型兼容性。对于根据零件实例装配 (["Defining an assembly," Section 2.10.1](pt01ch02s10aus28.md)) 定义的模型，无法从结果文件中读取形状变化数据。

##### 从静态分析结果文件中读取形状变化

要根据先前静态分析的变形几何定义形状变化，请指定先前静态分析的结果文件和步骤。或者，您可以指定从中读取位移数据的增量编号。 （默认情况下，Abaqus 将从结果文件上指定步骤的最后一个可用增量中读取数据。）此外，您还可以读取指定节点集的形状变化数据。

| **输入文件用法：** |``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, FILE=*results file*, STEP=*step*, INC=*inc*, NSET=*name* ```|
| --- | --- |

##### 从特征值分析结果文件中读取形状变化

要根据先前特征值分析的振型定义形状变化，请指定先前特征频率提取或特征值屈曲预测分析的结果文件和步骤。或者，您可以指定从中读取特征向量数据的模式编号。 （默认情况下，Abaqus 将从结果文件中指定步骤可用的第一个特征向量中读取数据。）此外，您还可以读取指定节点集的特征模式数据。

| **输入文件用法：** |``` [*PARAMETER SHAPE VARIATION](../key/key-link.md#usb-kws-mparametershape), PARAMETER=*name*, FILE=*results file*, STEP=*step*, MODE=*mode*, NSET=*name* ```|
| --- | --- |

### 形状参数化和设计敏感性分析

为了使用 Abaqus/Design (["Design sensitivity analysis," Section 19.1.1](pt04ch19s01aus107.md)) 进行设计灵敏度分析，如果为参数形状变化指定的参数也指定为设计参数，则形状变化用于定义节点坐标和节点法线相对于设计参数的设计梯度。如果您希望对基本形状执行设计敏感性分析，则所有形状参数必须指定为零值。此外，如果参数形状变化定义中指定的*任何*参数也被指定为设计参数，则*所有*参数形状变化的参数必须指定为设计参数。

在壳和梁单元的 DSA 计算中，Abaqus 始终使用节点坐标的设计梯度来计算节点法线的设计梯度。要覆盖 Abaqus 计算的梯度，您必须提供节点法线作为节点定义的一部分，并使用参数形状变化设计法线的梯度。要规定独立于设计的法线，您必须明确提供零设计梯度。对于从结果文件中读取的形状变化，Abaqus 根据位移计算法线的梯度并忽略节点旋转。

对于梁单元，Abaqus 使用节点坐标的梯度和使用参数形状变化指定的![](../graphics/usb_eqn00150.gif)方向的梯度来计算梁横截面的![](../graphics/usb_eqn00149.gif)方向的设计梯度。您无法提供![](../graphics/usb_eqn00149.gif)方向的形状变化。 Abaqus 会忽略梁截面定义中或作为梁单元连接中的额外节点隐式提供的任何此类设计梯度。

如果定义形状变化的数据是在圆柱或球面坐标系中给出的，那么了解如何根据数据计算形状变化非常重要。此计算已在上一节中描述。

### 形状变化的可视化

只有在分析输入文件处理器处理参数化输入文件后，形状变化才能可视化。因此，在使用 Abaqus/CAE 可视化参数形状变化之前，至少必须执行数据检查运行。

与每个单独的形状参数相关的形状变化可以可视化为分析步骤零处的位移形状图。基本形状被解释为未变形形状，而通过将![](../graphics/usb_eqn00147.gif)形状变化添加到基本形状而生成的形状被解释为![](../graphics/usb_eqn00147.gif)位移形状。

添加到基本形状的所有形状变化的组合代表了分析的真实未变形形状。

### 使用 Abaqus/CAE 计算形状变化

Abaqus 脚本接口命令`_computeShapeVariations( )`提供计算形状变化的功能。使用该命令需要熟悉 Abaqus 脚本接口以及 Abaqus/CAE 中脚本的执行。["Design sensitivity analysis: overview," Section 14.1.1 of the Abaqus Example Problems Guide](../exa/exa-link.md#exa-dsa-overview)中描述并说明了必须遵循的程序。
