# 33.1.1 离散粒子单元

**产品：**Abaqus/Explicit

##### **参考**

- [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection)

### 概述

离散粒子单元：

- 用刚性球形表示单个粒子；
- 通常用于包含大量离散粒子单元相互作用以及与其他物体相互作用的分析；
- 只能用于显式动力分析；并且
- 必须只有一个节点。

### 典型应用

离散粒子单元（PD3D）适用于涉及砾石等不连续介质的仿真。关于离散单元法（DEM）及其典型应用的说明，见 ["离散单元法," 第 15.1.1 节](pt04ch15s01aus94.md)。

### 定义单元的截面属性

必须将离散截面定义与一组离散粒子单元关联。截面定义提供与 PD3D 单元相关的密度和粒子半径。

| **输入文件用法：** | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), ELSET=*element_set_name* ``` |
| --- | --- |

### 粒子几何、质量和转动惯量

PD3D 单元为球形，每个粒子具有均匀密度。粒子半径和密度在离散截面定义中指定。Abaqus 使用这些量计算 PD3D 单元的质量和转动惯量。每个离散单元本身很简单，但大量这类粒子和有限元组成的系统之间的相互作用可以模拟复杂现象，如 ["离散单元法," 第 15.1.1 节](pt04ch15s01aus94.md) 所述。

通常，分配给同一离散截面定义的所有粒子具有相同的粒径和密度；但也可以通过为半径和/或密度指定分布名称而不是标量值，在与同一离散截面定义关联的粒子之间指定这些量的变化。关于分布的讨论见 ["分布定义," 第 2.8.1 节](pt01ch02s08aus26.md)。例如，如果需要考虑两种粒径，最方便的方式可能是使用两个离散截面定义，并在每个定义中指定一种粒径。

| **输入文件用法：** | 使用以下选项定义粒子形状、密度和半径： |
| --- | --- |
|  | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), SHAPE=SPHERE, DENSITY=*particle density * *or a particle densities distribution table name* *particle radius or particle radii distribution table name* ``` |

### Alpha 阻尼

可以为 PD3D 单元定义与质量和转动惯量成比例的阻尼，类似于点质量和转动惯量单元的阻尼（见 ["点质量," 第 30.1.1 节](pt06ch30s01alm21.md) 和 ["转动惯量," 第 30.2.1 节](pt06ch30s02alm22.md)）。该阻尼作用于单个粒子相对于“地面”的平动和转动速度，并且独立于接触阻尼（接触阻尼作用于邻近粒子配对的相对速度）。与 Abaqus/Explicit 中其他类型的可变形有限元不同，PD3D 单元没有全局黏性阻尼。少量质量比例阻尼有助于降低大量接触条件反复开闭在解中产生的噪声。

| **输入文件用法：** | 使用以下选项指定质量和转动惯量比例阻尼： |
| --- | --- |
|  | ``` [*DISCRETE SECTION](../key/key-link.md#usb-kws-mdiscretesection), ALPHA=*damping factor, alpha* ``` |
