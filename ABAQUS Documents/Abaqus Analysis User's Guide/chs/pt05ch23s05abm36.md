概述

接头材料模型：
-旨在为包含高密度平行接头表面的材料提供一个简单的连续体模型，其中每个平行接头系统与特定方向相关联，例如沉积岩;
-假设与模型域中的特征尺寸相比，特定方位接头的间距足够接近，使得接头可以涂抹成连续的滑动系统；
-提供这些系统中的每个系统中的接头的开口或摩擦滑动（在这种情况下， “系统”是指在材料计算点处在特定方向上的接头取向） ；以及
-假设材料的弹性行为是各向同性和线性的，当一个点上的所有接头都闭合时（各向同性线性弹性行为必须包括在材料定义中；见["Defining isotropic elasticity" in "Linear elastic behavior," Section 22.2.1](pt05ch22s02abm02.md#usb-mat-clinearelastic-isotropic)）。

# # # #提高关节反复打开和关闭时的收敛性

当关节的重复打开和关闭使收敛变得困难时，可以通过防止关节打开来提高收敛性。在这种情况下，弹性刚度总是与关节相关联。当关节的打开和关闭仅限于模型的小区域时，它最有用。只有在指定接头方向时，才能防止接头打开，如下所述。

# # #非关联流程

如果在任何接头系统中![](../graphics/usb_eqn03616.gif)，无论是与接头表面还是散装材料相关联，该系统中的流量都是“非相关的。“这意味着材料刚度矩阵不是对称的。因此，分析步骤（["Defining an analysis," Section 6.1.2](pt03ch06s01abo05.md)）应使用非对称矩阵解方案，特别是当模型的大区域预计将塑性流动并且![](../graphics/usb_eqn03300.gif)和![](../graphics/usb_eqn00177.gif)之间的差异较大时。 如果![](../graphics/usb_eqn03300.gif)和![](../graphics/usb_eqn00177.gif)之间的差异不大，则矩阵的对称近似可以提供平衡方程的可接受收敛速率，从而降低总体解成本。因此，当定义连接材料行为时，不会自动调用非对称矩阵解方案。

Elements

接头材料模型可用于ABAQUS/Standard中的平面应变、广义平面应变、轴对称和三维固体（连续体）元件。该模型不能用于假定应力状态为平面应力的元素（平面应力、壳体和膜元素）。
