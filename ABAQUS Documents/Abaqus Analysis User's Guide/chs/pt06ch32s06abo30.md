# 32.6.1 垫片单元：概述

Abaqus/Standard 提供垫片单元库，用于模拟垫片行为。

### 概述

垫片建模包括：

- 选择合适的垫片单元类型（["选择垫片单元," 第 32.6.2 节](pt06ch32s06alm47.md)）；
- 在有限元模型中包含垫片单元（["在模型中包含垫片单元," 第 32.6.3 节](pt06ch32s06alm48.md)）；
- 定义垫片的初始几何（["定义垫片单元的初始几何," 第 32.6.4 节](pt06ch32s06alm49.md)）；以及
- 使用材料模型（["使用材料模型定义垫片行为," 第 32.6.5 节](pt06ch32s06alm50.md)）或垫片行为模型（["直接使用垫片行为模型定义垫片行为," 第 32.6.6 节](pt06ch32s06alm51.md)）定义垫片行为。

### 垫片单元的用途

垫片可由多种方式和多种材料构成。某些垫片由若干层预成形金属组成，可能带有薄的弹性体涂层或弹性体嵌件（见 [图 32.6.1--1](pt06ch32s06abo30.md#egasket-typical)）。其他垫片则使用塑料和弹性体嵌件。

**图 32.6.1--1** 由若干层预成形金属组成的典型垫片。

![](../graphics/egasket-typical-nls.png)

垫片通常很薄，并作为结构部件之间的密封部件。垫片经过仔细设计，以在其厚度方向（垫片的薄方向）提供合适的压力-闭合行为，从而在部件因热载荷和机械载荷发生变形时保持密封作用。使用可用材料库中的实体连续体单元很难模拟垫片的厚度方向行为。因此，Abaqus/Standard 提供多种垫片单元，它们具有专门为研究垫片而设计的厚度方向行为。

垫片行为模型独立于材料库中的模型，并假定厚度方向行为、横向剪切行为和膜行为相互解耦（详情见 ["直接使用垫片行为模型定义垫片行为," 第 32.6.6 节](pt06ch32s06alm51.md)）。对于这些特殊行为模型不容易处理的垫片行为，例如必须考虑耦合行为或厚度方向拉伸行为时，Abaqus/Standard 允许垫片单元使用内置或用户定义材料模型，从而提供一种灵活替代方案（详情见 ["使用材料模型定义垫片行为," 第 32.6.5 节](pt06ch32s06alm50.md)）。

### 垫片单元的空间表示

[图 32.6.1--2](pt06ch32s06abo30.md#egasket-spatial-rep) 展示了用于定义垫片单元的关键几何特征。垫片单元由两个相隔一定厚度的表面组成。沿垫片厚度方向测量的底面和顶面相对运动，用于量化垫片单元的厚度方向（局部 1 方向）行为。在垂直于厚度方向的平面内测量的底面和顶面相对位置变化，用于量化垫片单元的横向剪切行为。单元中面（位于底面和顶面中间的表面）的拉伸和剪切用于量化垫片单元的膜行为。

### 积分点处定义的局部行为方向

垫片单元积分点处定义的厚度方向构成局部 1 方向。横向剪切行为定义在局部 1-2 和 1-3 平面中。膜行为定义在 2-3 平面中。对于节点只有一个自由度的单元，不定义局部 2 和 3 方向，因为这类单元只考虑垫片的厚度方向行为。局部方向用于指定垫片行为，并用于输出描述垫片当前变形状态的所有量。Abaqus/Standard 默认计算局部方向。对于某些单元类型，也可以由用户定义局部方向。

#### 默认局部方向

Abaqus/Standard 按 ["定义垫片单元的初始几何," 第 32.6.4 节](pt06ch32s06alm49.md) 中说明的方法计算局部 1 方向。

对于二维和轴对称垫片单元，局部 2 方向定义为使局部 1 和 2 方向的叉乘给出平面外方向（见 [图 32.6.1--3](pt06ch32s06abo30.md#egasket-2d-axi-local-dir)）。

**图 32.6.1--3** 二维和轴对称垫片单元的局部方向。

![](../graphics/egasket-2d-axi-local-dir.png)

对于三维面积单元和三维 link 单元，局部 2 和 3 方向垂直于局部 1 方向（见 [图 32.6.1--4](pt06ch32s06abo30.md#egasket-3d-local-dir)），并按 Abaqus 对空间表面局部方向的标准约定定义（见 ["约定," 第 1.2.2 节](pt01ch01s02aus02.md)）。

**图 32.6.1--4** 三维面积和三维 link 垫片单元的局部方向。

![](../graphics/egasket-3d-local-dir-nls.png)

对于三维线单元，局部 2 方向通过将单元中面的切线投影到垂直于局部 1 方向的平面上得到（见 [图 32.6.1--5](pt06ch32s06abo30.md#egasket-3d-line-local-dir)）。局部 3 方向随后由局部 1 和 2 方向的叉乘得到。

**图 32.6.1--5** 三维线垫片单元的局部方向。

![](../graphics/egasket-3d-line-local-dir-nls.png)

#### 指定局部方向

可以按 ["定义垫片单元的初始几何," 第 32.6.4 节](pt06ch32s06alm49.md) 中说明的方法定义局部 1 方向。对于考虑横向剪切和膜变形的三维面积单元和三维 link 单元，可以使用局部方向（["方向," 第 2.2.5 节](pt01ch02s02aus15.md)）定义局部 2 和 3 方向。

|  | ``` [*GASKET SECTION](../key/key-link.md#usb-kws-mgasketsection), ELSET=*name*, ORIENTATION=*name* ``` |

| **Abaqus/CAE 用法：** | Property 模块：**Assign** → **Material Orientation** |

### 允许使用垫片单元的过程

垫片单元可用于静态、静态扰动、准静态、动力和频率分析。但是，垫片单元假定没有质量；因此，不能为垫片单元定义密度。
