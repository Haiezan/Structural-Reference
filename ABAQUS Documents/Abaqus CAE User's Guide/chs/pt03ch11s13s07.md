# 11.13.7 通过节距控制旋转特征的横截面







当您创建不带节距的旋转特征时，绘制的轮廓将围绕旋转轴与草图之间的半径所描述的圆形路径进行扫掠。旋转特征的横截面是草图；横截面始终平行于旋转轴并垂直于圆形路径。

当您在旋转特征中包含螺距时，草图路径将变为螺旋形而不是圆形。您可以选择保持草图与旋转轴平行，也可以选择垂直于螺旋路径旋转草图。

要使绘制的轮廓垂直于带螺距的旋转特征的螺旋路径，请在 **编辑旋转** 对话框中打开 **扫描草图垂直于路径**。当 Abaqus/CAE 创建旋转特征时，它会旋转草绘轮廓，使其垂直于起点处的旋转路径。在整个特征创建过程中，轮廓与路径保持垂直。无论螺距值如何，横截面都将与绘制的轮廓相匹配。使用此选项，您可以创建螺旋弹簧或其他特征，其中路径的横截面是您绘制的轮廓。

如果不选择 **垂直于路径的扫描草图**，绘制的轮廓将保持与旋转轴平行，并且旋转特征的横截面将与轮廓不同。随着节距值的增加，轮廓和横截面之间的差异将会增加。例如，如果没有节距，则圆形草绘轮廓将创建圆形横截面。如果增加螺距，横截面将变得越来越椭圆。您可以创建螺纹或其他特征，其中平行于旋转轴的横截面是您绘制的轮廓。
