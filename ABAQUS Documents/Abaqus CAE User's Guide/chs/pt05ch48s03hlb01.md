# 48.3.1 选择在获取数据的位置的路径





Abaqus/CAE 以 X–Y 数据对的形式获取沿路径的结果。数据对 X 值定义获取数据的模型位置；Y 值是那些位置的分析结果。Abaqus 在计算数据对时仅考虑当前显示组中的实体。

Abaqus 根据您为路径指定的点来计算数据对 X 值。您可以选择仅根据您指定的点形成 X 值，或者额外包括路径与模型相交的所有位置。（相交发生在路径穿过单元面、单元边、表面面或表面边的地方。）

您还可以选择 Abaqus 是否将构成路径的点解释为未变形或变形模型形状上的位置。对于节点列表路径，构成路径的节点标签同样适用于未变形和变形的模型形状。但是，点列表路径坐标在空间中是固定的，与模型无关。例如，与未变形形状上节点位置重合的坐标可能与变形形状上的任何位置都不重合。使用点列表路径坐标和变形模型获取的值受变形比例因子的影响。对于与模型不重合的点，Abaqus 不会形成数据对。

**选择在获取数据的位置的路径：**

1. 找到 **沿路径的 XY 数据（XY Data from Path）** 选项。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**创建（Create）****。从出现的对话框中，选择 **路径（Path）**；然后点击 **继续（Continue）**。出现 **沿路径的 XY 数据（XY Data from Path）** 对话框。
2. 点击 **路径（Path）** 箭头选择要获取数据的路径。当前视口中的模型绘图会更改以高亮显示您选择的路径。
3. 点击 **未变形（Undeformed）** 或 **变形（Deformed）** 来选择 Abaqus 是将构成路径的点解释为未变形还是变形模型形状上的位置。
4. 从 **点位置（Point Locations）** 选项中，选择以下之一：- 选择 **路径点（Path points）** 仅在构成路径的点处获取数据。您可以通过切换 **包括相交点（Include intersections）** 来在路径与模型相交的位置以及构成路径的点处获取 X–Y 数据。- 选择 **均匀间距（Uniform spacing）** 在沿路径的规则插值间隔处获取数据，但不一定在路径定义包含的点处。您可以更改 **间隔数（Number of intervals）** 字段中指定的值来增加或减少沿路径获取插值数据的位置数。
5. 要评估和显示数据，请点击 **绘图（Plot）**。X–Y 绘图出现在当前视口中。该绘图表示您在对话框中配置的数据，无论您是否点击 **另存为（Save As）** 来保存它，Abaqus 都将其视为临时数据。
6. 要保存您配置的数据，请点击 **另存为（Save As）**。**注意：**要绘制您保存的 X–Y 数据对象，请从主菜单栏中选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**XY 数据（XY Data）**![](../graphics/images/arrow.gif)**绘图（Plot）****，并从弹出菜单中选择 X–Y 数据对象。
7. 完成后，点击 **取消（Cancel）** 关闭对话框。
