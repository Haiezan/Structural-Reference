# 11.18.2 创建新零件







从主菜单栏中选择****零件**![](../graphics/images/arrow.gif)**创建****，在当前视口中创建一个新零件。

一个模型可以包含多个部分；每个零件都存在于局部坐标系中，您可以使用“装配”模块创建零件的实例，并在全局坐标系中相对于彼此定位这些实例。创建零件时，您可以为零件命名并选择其类型、建模空间、基本特征和大致尺寸；然后绘制零件基本特征的轮廓草图。

**创建新零件：**

1. 从主菜单栏中，选择****零件**![](../graphics/images/arrow.gif)**创建****。将出现 **创建零件** 对话框。有关详细信息，请参阅["Using the **Create Part** dialog box," Section 11.19](pt03ch11s19.md)。 **提示：**您还可以使用部件模块工具箱中的![](../graphics/ico_part.png)工具创建零件。有关部件模块工具箱中工具的图表，请参阅["Using the Part module toolbox," Section 11.17](pt03ch11s17.md)。
2. 键入零件的名称。有关命名 Abaqus/CAE 对象的信息，请参阅["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md)。您可以在创建零件后对其进行重命名。
3. 选择新零件的建模空间、类型、基本特征和大致尺寸。有关详细信息，请参阅["How is a part defined in Abaqus/CAE?," Section 11.4](pt03ch11s04.md)。 **注意：**要更改零件的建模空间或类型，您必须使用模型树来编辑零件。有关更多信息，请参阅["How is a part defined in Abaqus/CAE?," Section 11.4](pt03ch11s04.md)。
4. 单击“**继续**”关闭“**创建零件**”对话框。草绘器启动，**草图**网格出现在当前视口中。如果要创建三维旋转实体或壳，Abaqus/CAE 会在草图的 *Y* 轴上显示一条垂直构造线，该轴用作旋转轴。您可以在此构造线的任一侧绘制草图，但草图不得穿过构造线。如果要创建轴对称零件，Abaqus/CAE 会在草图左侧显示一条垂直构造线，用作旋转轴。您必须在此构造线的右侧绘制草图。
5. 使用草绘器绘制基础特征的二维轮廓。有关详细信息，请参见[Chapter 20, "The Sketch module](pt03ch20.md)。” 如果要构造扫描零件，则必须首先绘制扫描路径并退出草绘器。 **草图** 随后将自动重新启动，您可以绘制要扫描的轮廓。
6. 完成基本特征的草图绘制后，单击鼠标按钮 2 退出当前的 **草图** 工具。
7. 在提示区域，单击“**完成**”退出草绘器。如果基础特征是三维实体或壳拉伸，则会出现“**编辑基础拉伸**”对话框。您必须使用 **深度** 字段输入挤出轮廓的距离。您还可以选择 **Twist** 或 **Draft** 来修改 Abaqus/CAE 将创建的拉伸形式。如果基础特征是三维旋转实体或壳，则必须输入轮廓旋转的角度。 Abaqus/CAE 退出草绘器并在当前视口中显示新零件。
8. 如有必要，请使用部件模块工具向基本特征添加附加特征。有关详细信息，请参阅["What is feature-based modeling?," Section 11.3](pt03ch11s03.md)。
