# 59.7.2 编辑基于边缘的图案创建的附着点







您可以编辑由基于边缘的阵列创建的附着点。

**要编辑由基于边缘的图案创建的附着点：**

1. 从主菜单栏中，选择****功能**![](../graphics/images/arrow.gif)**编辑****。 **提示：**您还可以通过单击模块工具箱中的![](../graphics/ico_featureEdit.png)工具来编辑功能。
2. 选择要修改的附着点特征。您可以直接从当前视口或模型树中选择特征。将出现 **编辑附着点与边特征的偏移** 对话框。
3. 执行以下任一操作来指定沿边的点： - 选择“**按数字**”以定义沿边选择的附着点数量。 Abaqus/CAE 自动沿边缘间隔点。 1. 输入沿边缘的点数。 2. 输入第一个附着点相对于起点的所需偏移量。 3. 输入最后一个连接点相对于终点的所需偏移量。 - 选择**按间距**来定义附着点之间的距离。 1. 输入点之间的间距。 2. 输入第一个附着点相对于起点的所需偏移量。 3. 指定沿边缘的点数，或选择 **自动调整** 让 Abaqus/CAE 确定附着点的数量。如果指定点数，Abaqus/CAE 将仅创建起点和终点之间的点；任何延伸超过终点的点都不会被创建。
4. 如果需要，可以通过指定以下内容来定义连接点的行图案： - 输入图案中的行数。值为 1 会创建单行附着点。 - 输入间距值以定义图案行之间的距离。 - 输入图案第一行距边缘的偏移距离。值 0 将在边缘上创建图案的第一行。
5. 如果需要，在编辑过程完成后，切换 **Regenerate on OK** 以控制模型重新生成。
