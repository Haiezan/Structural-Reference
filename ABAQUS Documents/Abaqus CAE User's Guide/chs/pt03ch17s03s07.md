# 17.3.7 网格细化







网格模块提供了一组允许您细化网格的工具。
- 您可以使用分区工具集将几何区域划分为更小的区域。生成的分区引入了您可以播种的新边；因此，您可以结合分区和播种来获得对网格生成过程的额外控制。您还可以使用分区来创建可以分配不同单元类型的区域。例如，您可能希望将简化集成单元分配给模型的某些部分，并将完全集成单元分配给其他部分。有关详细信息，请参阅[Chapter 70, "The Partition toolset](pt06ch70.md)。”
- 在某些情况下，几何体包含细节，例如非常小的面和边缘。虚拟拓扑工具集允许您通过将小面与相邻面组合或将小边与相邻边组合来删除这些小细节。引入虚拟拓扑是创建干净、结构良好的网格的便捷方法。有关详细信息，请参阅[Chapter 75, "The Virtual Topology toolset](pt06ch75.md)。”
- 您可以使用“编辑网格”工具集对网格进行细微调整。有关更多信息，请参阅["What can I do with the Edit Mesh toolset?," Section 64.1](pt06ch64s01.md)。
