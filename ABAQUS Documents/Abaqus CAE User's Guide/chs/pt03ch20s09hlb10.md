# 20.9.10 自定义草绘器中约束的使用







[Constraints](pt03ch20s12hlb03.md)创建控制草图几何形状的非数字关系。当您编辑草图时，Abaqus/CAE 会应用求解方法以避免违反约束。当实体为[dragged](pt03ch20s14hlb01.md)或使用其他草绘工具进行编辑时，有多种方法可供使用。

**注意：**约束求解方法不会阻止草图修改工具覆盖应用的约束。例如，您可以使用 **平移** 工具来移动具有 **固定** 约束的线。

此外，您可以使用 **显示约束** 选项来隐藏约束并减少草图中的混乱，并使用 **在实体创建期间添加约束** 选项来添加草图绘制过程中隐含的约束。您可以选择线性和角度公差以及可以使用 **自动约束** 工具自动创建的约束类型。

约束自定义选项适用于整个 Abaqus/CAE 会话。它们不与单独的草图或当前模型一起保存。

**要在草绘器中自定义约束的使用：**

1. 从草绘器工具箱底部，选择草绘器自定义工具![](../graphics/ico_sketchOptions-nls.png)。将出现 **草绘器选项** 对话框。
2. 单击“**约束**”选项卡。
3. 选择在拖动草图实体或其他类型编辑期间使用的所需约束求解方法。可以选择以下选项： **标准** 这是用于编辑的默认解决方案方法。 Abaqus/CAE 以最小量移动或修改几何体以满足约束和尺寸。 Abaqus/CAE 还尝试在移动过程中保持未涉及几何形状的刚体外观。 **加权** Abaqus/CAE 按最小量移动或修改几何体以满足约束和尺寸。移动过程中不考虑未涉及几何体的刚体外观。 **最小移动** 这是拖动的默认解决方法。 Abaqus/CAE 移动最小数量的实体以满足约束。 **松弛** Abaqus/CAE 使用数值求解技术移动几何体，以最小化移动的平方和。与其他方法相比，此求解方法会移动更多的几何图形。
4. 如果需要，切换 **显示约束**。当 **显示约束** 启用时，Abaqus/CAE 在草图中显示约束符号。
5. 如果需要，切换 **在实体创建期间添加约束**。当**在实体创建期间添加约束**启用时，Abaqus/CAE 将添加草图绘制过程中隐含的约束。例如，如果您正在绘制一条线，其预选符号指示 Abaqus/CAE 将应用水平约束，并且 **在实体创建期间添加约束** 处于启用状态，则水平约束将添加到该线上。
6. 在页面的下部，切换可与 **自动约束** 工具一起使用的约束类型。 （有关详细信息，请参阅["Automatically constraining a sketch," Section 20.12.1](pt03ch20s12hlb01.md)。）
7. 选择所需的自定义选项后，单击“**确定**”应用更改并关闭“**草绘器选项**”对话框。
