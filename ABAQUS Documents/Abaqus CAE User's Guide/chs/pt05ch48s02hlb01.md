# 48.2.1 创建或编辑节点列表路径





路径是您通过在模型中指定一系列点来定义的线。在节点列表路径中，所有点都是节点位置。有关在路径中包含节点以外位置的信息，请参阅 ["在模型中创建路径，" 第 48.2 节"](pt05ch48s02.md)。

您可以通过将部件实例名称、节点标签和节点标签范围输入表格来创建节点列表路径。您可以使用以下过程中描述的编辑技术直接从视口中选择节点来创建路径。或者，您可以直接在表格中输入值来创建节点列表路径。如果不知道模型中的部件实例名称，请使用查询工具集来查询视口中的节点。为了帮助确定感兴趣的节点标签，您可能需要在创建路径之前生成带有节点符号和节点编号可见的模型绘图。有关更多信息，请参阅 ["自定义模型标签，" 第 55.5 节"](pt05ch55s05.md)。

**创建或编辑节点列表路径：**

1. 找到路径创建和编辑选项。**创建新路径：** 1. 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**创建（Create）****。出现 **创建路径（Create Path）** 对话框。2. Abaqus 在 **名称（Name）** 文本字段中显示路径的默认名称。为了提供更有意义的名称，用您选择的名称替换此默认值（包括您希望的任何空格）。3. 在 **创建路径（Create Path）** 对话框中，点击 **节点列表（Node list）**；然后点击 **继续（Continue）**。**编辑现有路径：** - 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**编辑（Edit）****。从出现的菜单中，选择您要编辑的路径。**提示：**您也可以使用 **路径管理器（Path Manager）** 来编辑路径。从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**路径（Path）**![](../graphics/images/arrow.gif)**管理器（Manager）**** 来显示管理器。选择您要编辑的路径，然后点击管理器右侧的按钮上的 **编辑（Edit）**。出现 **编辑节点列表路径（Edit Node List Path）** 对话框。
2. 要指定沿路径的节点，请使用以下任一技术：**直接从视口中选择节点：** 1. 点击 **之前添加（Add Before）** 或 **之后添加（Add After）** 开始进行选择。Abaqus/CAE 提示您 **选择要插入路径的节点（Select nodes to be inserted into the path）**，并关闭 **编辑节点列表路径（Edit Node List Path）** 对话框。2. 从视口中选择一个或多个节点（有关更多信息，请参阅 [第 6 章，"在视口中选择对象"](pt01ch06.md)）。如果所需节点存在于多个部件实例中，请关闭 **选择最接近屏幕的实体（Select the Entity Closest to the Screen）** 工具（位于 **选择（Selection）** 工具栏中）。如果进行模糊选择，部件实例名称和节点标签将以下形式出现在视口左下角：`高亮显示节点 *实例*.*节点*（Highlighting Node *Instance*.*Node*）`。使用 **下一个（Next）** 和 **上一个（Previous）** 按钮循环浏览可能的选项，然后点击 **确定（OK）** 确认您的选择。Abaqus 在您选择时高亮显示节点，并显示节点标签和路径连通性。**注意：**您只能选择当前显示组中的节点。3. 完成后，在提示区域点击 **完成（Done）**。Abaqus/CAE 显示 **编辑节点列表路径（Edit Node List Path）** 对话框；路径中的部件实例和节点列在路径定义表中。4. 如果需要，重复前述步骤添加其他点。**之前添加（Add Before）** 和 **之后添加（Add After）** 按钮决定您的选择将被添加到路径定义表中当前高亮行的什么位置。**直接在表格中输入节点标签：** 1. 在 **路径定义（Path Definition）** 表中，使用标准键盘和鼠标编辑技术来插入、修改或删除部件实例名称和节点标签表达式。如果不知道模型中的部件实例名称，请使用查询工具集来查询视口中的节点。有关特殊的表编辑选项或从 ASCII 文件读取数据，请点击鼠标按钮 3。（有关更多信息，请参阅 ["输入表格数据，" 第 3.2.7 节"](pt01ch03s02s07.md)。）节点标签表达式是以下之一：- 单个节点标签；例如，5。- 节点范围，用起始和结束节点标签指定，可选择后跟增量；例如，5:10 或 5:10:2。**注意：**如果从 ASCII 文件读取节点标签，必须从第一列开始读取表格值，每行必须包含部件实例名称和节点标签表达式。部件实例名称区分大小写，必须用引号括起来，格式如下： ``` `"*实例名称*"` ``` 2. 在每个节点标签表达式后，按 **[Enter]**。Abaqus 高亮显示路径中包含的所有节点，并显示节点标签和路径连通性。
3. 完成后选择节点后，点击 **确定（OK）**。如果创建了新路径，Abaqus/CAE 会将路径名称添加到 **路径管理器（Path Manager）** 列表中。默认情况下，Abaqus/CAE 会保存您在会话期间指定的路径。如果您想在后续会话中保留路径，请将其保存到 XML 文件、模型数据库或输出数据库；有关更多信息，请参阅 ["管理会话对象和会话选项，" 第 9.9 节"](pt02ch09s08.md)。
