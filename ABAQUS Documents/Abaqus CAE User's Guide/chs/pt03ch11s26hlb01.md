# 11.26.1 创建放样剖面







放样截面表示放样特征在沿放样路径的特定点处将具有的形状。创建放样特征至少需要两个截面。您可以通过从当前视口中的零件的现有边拾取来创建放样剖面。

**创建阁楼剖面：**

1. 从 **编辑放样** 对话框中，选择 **插入之前** 或 **插入之后** 以创建第一个放样剖面。
2. 选择定义放样截面的边线。 **[Shift]****+单击**其他边将其添加到您的选择中，并**[Ctrl]****+单击**选定的边以取消选择它。选定的边以红色突出显示，并将保持突出显示状态，直到放样特征完成。放样剖面可以由当前零件的任何边组成，并且必须满足以下条件： - 对于实体​​放样或切割放样，剖面必须闭合，并且第一个和最后一个剖面必须是平面的。 - 对于外壳阁楼，如果其中一部分开放，则所有部分都必须开放；而且，如果关闭某一部分，则所有部分都必须关闭。 - 该部分必须是一个连续的循环，没有分支。 - 在一个放样剖面中使用的边不能在同一放样的另一剖面中使用。所有壳放样截面以及实体放样和切割放样的中间截面都可以是非平面的。要创建非平面截面，您可以选择使用样条线创建的边或选择现有非平面的边。
3. 在提示区域中，单击“**完成**”表示您已完成选择边线来定义放样截面。 Abaqus/CAE 返回到 **编辑放样** 对话框，并且该截面出现在 **截面** 列表中。
4. 重复步骤 2 和 3 以定义所有放样截面。使用 **插入之前** 和 **插入之后** 按钮指示您希望下一个放样剖面在列表中相对于当前突出显示的放样剖面出现的位置。列表中突出显示的部分也会在视口中以洋红色突出显示，以将其与其他放样部分区分开来。放样截面按照创建它们的顺序进行编号，但 Abaqus/CAE 按照 **截面** 列表中显示的顺序创建放样特征。您只能通过添加或删除放样剖面来更改列表的顺序。您无法编辑已完成的放样剖面。
