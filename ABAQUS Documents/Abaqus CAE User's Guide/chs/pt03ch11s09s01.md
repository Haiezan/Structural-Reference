# 11.9.1 可靠的功能







要创建实体特征，请从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体****或选择部件模块工具箱中的实体工具之一。绘制初始轮廓草图后，您可以执行以下操作之一来创建特征：
- 要创建拉伸实体特征，请将轮廓拉伸指定距离 (*d*)，如[Figure 11--15](pt03ch11s09s01.md#prt-solid-extrude)中所示。 **图 11--15** 拉伸实体特征。![](../graphics/prt-solid-extrude-nls.png)此外，您可以对拉伸应用拔模或扭转，如[Figure 11--16](pt03ch11s09s01.md#prt-solid-draft)中所示。 **图 11--16** 具有拔模斜度和扭曲的拉伸实体特征。![](../graphics/prt-solid-draft-nls.png)您可以定义带拔模的挤出的拔模角或扭转中心以及带扭转的挤出的节距（发生 360 度扭转的挤出距离）。从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**拉伸****来创建此类特征。
- 要创建实体放样特征，请将形状从初始放样截面过渡到不同形状或方向的末端截面。 Abaqus/CAE 使用相切约束、中间截面和路径曲线确定起始截面和结束截面之间的形状。[Figure 11--17](pt03ch11s09s01.md#prt-solid-loft)中显示了一个简单的放样（只有两个放样截面、没有相切约束和一条直线路径）。从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**放样****来创建此类特征。 **图 11--17** 实体放样特征。![](../graphics/prt-solid-loft-nls.png)
- 要创建旋转实体特征，请将轮廓旋转指定角度 (![](../graphics/usi_eqn00007.gif))。构造线用作旋转轴，如[Figure 11--18](pt03ch11s09s01.md#prt-solid-rev)所示。 **图 11--18** 旋转实体特征。![](../graphics/prt-solid-rev-nls.png)此外，您还可以输入螺距值 (*p*)，以在轮廓旋转时沿旋转轴平移轮廓；[Figure 11--19](pt03ch11s09s01.md#prt-solid-revpitch)显示了带俯仰的立体旋转 360 度。从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**旋转****来创建此类特征。 **图 11--19** 带节距 (*p*) 的 360 度旋转实体特征。![](../graphics/prt-solid-revpitch-nls.png)- 要创建扫掠实体特征，请沿指定路径扫掠轮廓，如[Figure 11--20](pt03ch11s09s01.md#prt-solid-sweep)中所示。从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**扫描****来创建此类特征。有关更多信息，请参见["Defining the sweep path and the sweep profile," Section 11.13.8](pt03ch11s13s08.md)。 **图 11--20** 扫掠实体特征。![](../graphics/prt-solid-sweep-nls.png)

您可以使用任何实体工具将实体特征添加到在三维建模空间中创建的可变形或离散零件。您无法将实体特征添加到二维或轴对称零件。
