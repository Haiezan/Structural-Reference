# 3.4.11 使用管理器对话框复制步骤相关对象







您可以使用管理器对话框中的 **复制** 按钮来复制与步骤相关的对象，例如载荷、边界条件、交互、预定义字段、输出请求或自适应网格约束。您可以将对象从任何步骤复制到任何其他有效步骤。

**要复制依赖于步骤的对象：**

1. 打开相应的管理器，如["Managing objects using manager dialog boxes," Section 3.4.9](pt01ch03s04hlb01.md)中所述。
2. 通过单击相应的表格单元格来选择要复制的对象。
3. 单击“**复制**”。将出现相应的 **复制 *对象*** 对话框。
4. 输入新对象的名称。
5. 从 **步骤** 列表中选择目标步骤。默认情况下，对象被复制到创建它的步骤中。
6. 单击“**确定**”。对象被复制到选定的目标步骤。

如果目标步骤类型与最初创建对象的步骤类型不同，Abaqus/CAE 可能需要修改对象以使其与新步骤兼容。例如，如果在静态（一般）步骤中创建载荷并将其复制到稳态动态（线性扰动）步骤中，Abaqus/CAE 会向载荷值添加虚部。相反，如果将稳态动态步骤的载荷复制到静态步骤，则虚部将被删除。

此外，当您将对象复制到创建该对象的步骤以外的步骤时，在其传播状态之一中对原始对象所做的任何修改都将在副本中被忽略。

Abaqus/CAE 会阻止您将对象复制到对象类型无效的步骤类型中。 Abaqus/CAE 还阻止您复制抑制的对象或复制到抑制的步骤。
