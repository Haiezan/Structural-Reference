# 18.6.4 配置大小优化任务







大小调整优化是一种灵活的、基于敏感性的优化，允许您将一系列约束和目标函数应用于模型。您可以使用优化任务编辑器来自定义大小优化的各个方面。要找到编辑器，请从主菜单栏中选择****任务**![](../graphics/images/arrow.gif)**编辑**![](../graphics/images/arrow.gif)***优化任务名称*****。

### 配置基本设置

**配置基本设置：**

1. 在优化任务编辑器中，单击“**基本**”选项卡。
2. 选择是否冻结载荷或边界条件区域。建议您冻结应用了规定条件的区域，因为您不希望这些区域在优化过程中被删除。冻结这些区域可以稳定优化，并且通常会导致迭代次数显着减少。

### 配置厚度设置

**配置厚度设置：**

1. 在优化任务编辑器中，单击“**厚度**”选项卡。
2. 选择**厚度更新策略**。此设置控制优化模块在使用移动渐近线方法进行优化期间更新设计单元的壳厚度的速率。在大多数情况下，您应该接受默认设置（**正常**）。但是，如果设计响应非常敏感并且您在满足约束方面遇到问题，则可能需要更保守的速率，这需要更多的优化迭代。选择激进的速率可能会导致优化不稳定或阻止优化收敛于解决方案。
3. 输入**每个设计周期的最大变更**。此设置控制每个设计周期期间壳单元厚度变化的限制。

### 配置扰动设置

**配置扰动设置：**

1. 在优化任务编辑器中，单击 **扰动** 选项卡。
2. 输入要跟踪的本征模数。默认值为 5，这意味着优化模块将尝试跟踪五个最低特征频率。在某些情况下，优化迭代期间会出现许多局部低频特征模式，这会导致大量模式需要跟踪并降低性能。您可以通过在第一次优化迭代中将特征频率的下限选择为感兴趣的特征频率的 25% 来避免跟踪大量模式。如果您的设计响应将使用 Kreisselmaier-Steinhauser 公式来评估特征频率，则不需要模式跟踪。您的 Abaqus 模型必须包含至少您正在跟踪的特征频率数量的输出请求。
3. 选择优化模块应跟踪特征模式的区域。

### 配置收敛选项

**配置收敛选项：**

1. 在优化任务编辑器中，单击“**收敛**”选项卡。
2. 指定**收敛标准**。以下选项允许您指定大小优化的收敛标准： **指定何时开始检查收敛** 您可以指定优化模块开始检查两个收敛标准的迭代。优化将始终持续至少直到达到该值。默认值为 4。 **指定要检查的收敛标准** 您可以指定当满足其中一个收敛标准或同时满足两个标准时是否应结束优化。默认值是必须满足两个条件。 **基于优化函数变化的收敛** 您可以指定优化将根据目标函数从一次迭代到下一次迭代的变化而结束。默认值为 0.001。 **基于单元厚度变化的收敛** 单元厚度是尺寸优化的设计变量。您可以指定优化将根据从一次迭代到下一次迭代的单元厚度的平均变化来结束。默认值为 0.005。
