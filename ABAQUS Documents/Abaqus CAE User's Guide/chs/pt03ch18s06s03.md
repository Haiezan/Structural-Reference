# 18.6.4 配置尺寸优化任务

尺寸优化是一种灵活的、基于灵敏度的优化方法，允许您对模型施加一系列约束和目标函数。可以使用优化任务编辑器自定义尺寸优化的各个方面。要找到该编辑器，请从主菜单栏选择 ****Task**![](../graphics/images/arrow.gif)**Edit**![](../graphics/images/arrow.gif)***optimization task name*****。

涵盖以下主题：

- [配置基本设置](pt03ch18s06s03.md#configuring-basic-settings)
- [配置厚度设置](pt03ch18s06s03.md#configuring-the-thickness-settings)
- [配置扰动设置](pt03ch18s06s03.md#configuring-the-perturbation-settings)
- [配置收敛选项](pt03ch18s06s03.md#configuring-convergence-options)

### 配置基本设置

**配置基本设置：**

1. 在优化任务编辑器中，单击 **Basic** 选项卡。
2. 选择是否冻结载荷或边界条件区域。建议冻结施加了规定条件的区域，因为通常不希望这些区域在优化过程中被移除。冻结这些区域可以稳定优化过程，并且通常会显著减少迭代次数。

### 配置厚度设置

**配置厚度设置：**

1. 在优化任务编辑器中，单击 **Thickness** 选项卡。
2. 选择 **Thickness update strategy**。该设置控制优化模块在优化过程中使用移动渐近线方法更新设计单元壳厚度的速率。多数情况下应接受默认设置（**Normal**）。但是，如果设计响应非常敏感并且难以满足约束，可能需要更保守的速率，这会需要更多优化迭代。选择激进速率可能导致优化不稳定，或阻止优化收敛到解。
3. 输入 **Maximum change per design cycle**。该设置控制每个设计循环中壳单元厚度变化的限值。

### 配置扰动设置

**配置扰动设置：**

1. 在优化任务编辑器中，单击 **Perturbation** 选项卡。
2. 输入要跟踪的特征模态数量。默认值为 5，表示优化模块将尝试跟踪最低的 5 个特征频率。在某些情况下，优化迭代期间会出现许多局部低频特征模态，导致需要跟踪的模态数量增加并降低性能。可以将特征频率下限选为第一次优化迭代中目标特征频率的 25%，以避免跟踪过多模态。如果设计响应使用 Kreisselmaier-Steinhauser 公式评估特征频率，则不需要模态跟踪。Abaqus 模型必须包含至少与跟踪特征频率数量相同的输出请求。
3. 选择优化模块应跟踪特征模态的区域。

### 配置收敛选项

**配置收敛选项：**

1. 在优化任务编辑器中，单击 **Convergence** 选项卡。
2. 指定 **Convergence Criteria**。以下选项用于指定尺寸优化的收敛准则：可以指定优化模块从哪一次迭代开始检查两个收敛准则；可以指定满足任一准则即结束优化，还是必须同时满足两个准则；可以基于相邻迭代之间目标函数变化结束优化；也可以基于相邻迭代之间单元厚度平均变化结束优化。默认情况下，从第 4 次迭代开始检查收敛，两个准则都必须满足，目标函数变化默认值为 0.001，单元厚度变化默认值为 0.005。
