# 11.14.4 自相交检查







由于通过放样创建的特征非常复杂，因此可以使用一组测试来确保几何图形对于分析有效。您可以定义放样截面和路径，以使放样特征与其自身相交。具有自相交的放样特征作为制造零件是不切实际的，并且也很难或不可能进行网格划分和分析。

当您在 **特征选项** 对话框中打开 **执行自相交检查** 时，Abaqus/CAE 在创建放样特征时会测试自相交。如果放样的任何面与其他面相交，Abaqus/CAE 将显示一条错误消息，指出存在无效相交，并且不会创建放样特征。完成测试所需的时间根据您尝试创建的阁楼的复杂程度而有所不同。例如，如果放样的形状因截面而异，或者如果您定义了复杂的放样路径，则测试将显着增加创建放样特征所需的时间。如果您选择不包括测试，Abaqus/CAE 将创建放样特征，无论几何体是否有效。![](../graphics/images/black4rule.gif)有关相关主题的信息，请单击以下任意项目：-["What is feature-based modeling?," Section 11.3](pt03ch11s03.md)-["Adding a solid loft feature," Section 11.21.4](pt03ch11s21hlb04.md)-["Adding a shell loft feature," Section 11.22.4](pt03ch11s22hlb04.md)-["Creating a loft cut," Section 11.24.2](pt03ch11s24hlb02.md)-["What are self-intersection checks?," Section 65.3.4](pt06ch65s03s04.md)
