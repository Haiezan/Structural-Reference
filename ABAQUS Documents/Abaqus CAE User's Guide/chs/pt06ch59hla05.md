# 59.8 删除附着点和线







您可以从模型中删除附着点和线。删除的附着点或线在模型树中显示为零件或装配体下 **特征** 容器中的 **删除附着** 特征，具体取决于创建附着点或线的模块。您可以抑制或删除“删除附件”功能来恢复删除的附件点或线。在部件模块或属性模块中创建的附着点只能从这两个模块中删除；在装配模块或相互作用模块中创建的附着点和线只能从这两个模块中删除。

**要删除附着点和线：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**附件**![](../graphics/images/arrow.gif)**删除附件****。
2. 从当前视口中，选择要删除的附着点或线。您可以单击提示区域右侧的“**集合**”来删除之前定义的集合。
3. 如果您位于部件模块或属性模块中，请单击提示区域中的“**完成**”，表示您已完成选择要删除的附着点。选定的附着点将从模型中删除，并在零件下的模型树中创建 **删除附着** 特征。
4. 如果您位于相互作用模块或装配模块中，请执行以下操作： 1. 在提示区域中，单击 **完成** 以指示您已完成选择要删除的附着点或线。 2. 在出现的对话框中，执行以下操作之一： - 单击“**是**”继续并从模型中删除选定的附着点或线。 **删除附件** 特征在装配体下的模型树中创建。 - 单击“**否**”返回视口以修改要删除的选定附着点或线。 - 单击“**取消**”退出该过程，而不删除任何附着点或线。
5. 如果需要，选择要删除的其他附件功能。
6. 单击鼠标按钮 2 退出删除附件过程。
