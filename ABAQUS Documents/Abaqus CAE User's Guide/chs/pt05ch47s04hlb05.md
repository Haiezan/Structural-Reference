# 47.5.5 自定义 X-Y 绘图坐标轴刻度模式

### 47.5.5 自定义 X-Y 绘图坐标轴刻度模式

单击 **Scale** 选项卡可以自定义 *X* 轴和 *Y* 轴的刻度模式。刻度模式选项允许您控制任一坐标轴上显示的主刻度线和次刻度线数量。默认情况下，Abaqus/CAE 根据绘图中包含的数据点最小值和最大值计算坐标轴的最小值和最大值。

**自定义 X-Y 绘图坐标轴刻度模式：**

1. 找到 **Tick Mode** 选项。选择 ****Options**![](../graphics/images/arrow.gif)**XY Options**![](../graphics/images/arrow.gif)**Axis****；或单击可视化模块工具箱中 *X-Y* 绘图工具旁的 ![](../graphics/ico_xyAxisOptions.png)。在出现的对话框中单击 **Scale** 选项卡。**Tick Mode** 选项显示在 **Scale** 页底部。
2. 在 **X Axis** 或 **Y Axis** 字段中，选择一个或多个坐标轴。
3. 选择沿所选坐标轴计算主刻度间隔的方法。在 **Major** 选项中选择以下选项之一：**Automatic** 表示 Abaqus 自动计算所选坐标轴上的主刻度间隔，并取整为合理间隔；**By increment** 会启用 **Tick increment** 文本字段，可输入所选坐标轴上主刻度之间的增量；**By count** 会启用 **Number of ticks per decade** 字段，可增加或减少所选坐标轴最小值与最大值之间显示的主刻度总数。
4. 指定次刻度线数量。在 **Minor** ticks 选项中，单击 **Ticks per increment** 箭头（用于线性或分贝坐标轴）或 **Ticks per decade** 箭头（用于对数坐标轴），指定每个主刻度之间的次刻度数量。
5. 单击 **Dismiss** 关闭 **Axis Options** 对话框。
