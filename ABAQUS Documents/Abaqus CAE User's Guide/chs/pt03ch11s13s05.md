# 11.13.5 定义轴对称零件和旋转特征的旋转轴







创建轴对称零件并向零件添加旋转特征时，轮廓草图必须包含定义旋转轴的构造线。以下规则适用于草图和构造线：

**创建轴对称零件**

您可以通过从主菜单栏中选择****零件**![](../graphics/images/arrow.gif)**创建****来创建由壳或线以及对称轴定义的轴对称零件。当您创建轴对称零件时，Abaqus/CAE 允许您在模型中包含扭转自由度。

当您绘制零件的基本特征时，Abaqus/CAE 会在表示对称轴的草图的 *Y* 轴上显示一条垂直构造线。您必须仅在线的右侧绘制草图。你的草图可以触及这条线，但不能越过它。

您只能将壳和线特征添加到轴对称基础特征中。当您添加特征时，Abaqus/CAE 会显示原始草图和构造线，并且适用相同的规则 - 您无法删除此构造线，并且必须仅在其右侧绘制草图。

**创建旋转特征**

您可以通过从主菜单栏中选择****零件**![](../graphics/images/arrow.gif)**创建****来创建具有旋转实体或旋转壳体基础特征的三维零件。同样，您可以通过从主菜单栏中选择****形状**![](../graphics/images/arrow.gif)**实体**![](../graphics/images/arrow.gif)**旋转****、****形状**![](../graphics/images/arrow.gif)**外壳**![](../graphics/images/arrow.gif)**旋转****或****形状**![](../graphics/images/arrow.gif)**切割**![](../graphics/images/arrow.gif)**旋转****，向三维实体和外壳添加旋转实体、外壳和切口。

旋转特征的草图必须包含表示旋转轴的构造线。当您创建新的旋转零件时，Abaqus/CAE 会创建一条穿过草绘器网格原点的垂直构造线。如果需要，您可以删除该构造线并以新的角度和位置重新绘制它。相反，当您向现有零件添加旋转特征时，必须绘制表示旋转轴的构造线。您可以在构造线的右侧或左侧绘制草图。你的草图可以触及这条线，但不能越过它。如果完成的草图包含多条构造线，Abaqus/CAE 会提示您选择代表旋转轴的线。

当您绘制表示旋转轴的构造线时，可以通过从基础零件中选择基准轴（如果存在）来定位构造线。不能直接选择基准轴；您必须从基准轴的任一端选择一个点。您可以使用基准轴创建同心特征。

退出草绘器时，Abaqus/CAE 将打开一个对话框以完成旋转特征的定义。您输入轮廓旋转的角度和旋转方向，并且可以选择是否通过包括螺距来沿旋转轴平移轮廓。您还可以指定平移的方向。螺距值是轮廓在 360 度旋转期间平移的距离。螺距允许创建螺旋弹簧等零件和螺纹等零件细节。

如果要创建其中草绘轮廓是拉伸而不是旋转的复杂形状（例如扭索或斜齿轮），则可以在拉伸实体、壳体或切割特征中包含扭曲。有关所有可用特征类型的基本信息，请参见["What types of features can you create?," Section 11.9](pt03ch11s09.md)；有关拉伸特征的更多信息，请参见["Defining the extrusion distance," Section 11.13.1](pt03ch11s13s01.md)。![](../graphics/images/black4rule.gif)有关相关主题的信息，请单击以下任意项目：-["Adding an extruded solid feature," Section 11.21.1](pt03ch11s21hlb01.md)-["Adding a revolved solid feature," Section 11.21.2](pt03ch11s21hlb02.md)-["Adding an extruded shell feature," Section 11.22.1](pt03ch11s22hlb01.md)-["Adding a revolved shell feature," Section 11.22.2](pt03ch11s22hlb02.md)-["Creating an extruded cut," Section 11.24.1](pt03ch11s24hlb01.md)-["Creating a revolved cut," Section 11.24.3](pt03ch11s24hlb03.md)
