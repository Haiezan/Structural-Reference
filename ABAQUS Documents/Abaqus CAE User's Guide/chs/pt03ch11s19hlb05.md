# 11.19.5 设置新零件的大致尺寸







使用“**创建零件**”对话框底部的“**近似尺寸**”文本字段来设置新零件的近似尺寸。 Abaqus/CAE 使用您输入的尺寸来计算草绘器图纸的尺寸及其网格的间距。零件尺寸的近似值必须介于 10000 (104) 和 0.001 (103) 单位之间。 Abaqus/CAE 不使用特定单位，但整个模型中的单位必须保持一致。

当您退出 **创建零件** 对话框时，Abaqus/CAE 进入草绘器，并绘制基本特征的轮廓。草绘器显示带有覆盖网格的方形图纸，并调整图纸的尺寸以近似零件的尺寸。因此，草图的尺寸将与您正在创建的零件具有相同的数量级。

如果您随后编辑零件，Abaqus/CAE 仍会根据创建基本特征时使用的相同尺寸来确定草绘器图纸的尺寸。因此，您应该设置零件的大致尺寸以匹配成品零件的最大尺寸。如果您随后发现零件超出了草绘器图纸的尺寸，请使用 **草图** 自定义选项来增加图纸尺寸。

**设置新零件的大致尺寸：**

1. 在 **创建零件** 对话框底部的 **近似尺寸** 文本字段中键入新零件的近似尺寸。
2. 选择完选项后，单击“**继续**”关闭“**创建零件**”对话框。草绘器首先根据新零件的大致尺寸设置图纸尺寸和网格间距，然后绘制新零件基本特征的轮廓。
