# 54.8 报告数据值、最小值、最大值和总计





对于 X–Y 数据或场输出的表格报告，您可以包含或抑制 X–Y 数据或场输出值、列最小值和最大值摘要以及列总计。默认情况下，Abaqus/CAE 仅在 X–Y 数据报告中包含实际数据值。在场输出值报告中，默认情况下，Abaqus/CAE 包含数据值以及列摘要和总计。

对于探针值的表格报告，Abaqus/CAE 始终在报告中包含实际数据值。您可以包含或抑制任何有意义的列的列最小值和最大值摘要以及列总计。

**报告数据值、最小值、最大值和列总计：**

1. 找到 **数据（Data）** 或 **数据值（Data Values）** 选项。**对于 X–Y 数据或场输出的报告：** 从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**XY**** 或 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **设置（Setup）** 选项卡。**数据（Data）** 选项位于页面底部。**对于探针值的报告：** 从主菜单栏中，选择 ****工具（Tools）**![](../graphics/images/arrow.gif)**查询（Query）****，然后在出现的对话框中点击 **探针值（Probe values）**。出现 **探针值（Probe Values）** 对话框。将光标移到当前视口中的模型或 X–Y 绘图上探测，然后点击鼠标按钮 1 将感兴趣的值存储在对话框中。完成后，点击 **写入文件（Write to File）**。出现 **报告探针值（Report Probe Values）** 对话框；**数据值（Data Values）** 选项位于对话框底部。
2. 对于 X–Y 数据报告或场输出报告，您必须选择以下一个或多个项目包含在报告中：- 切换 **XY 数据（XY data）** 或 **场输出（Field output）** 分别包含实际的 X–Y 数据或场输出值。关闭此选项会导致报告仅由您请求的列总计或最小/最大数据组成。- 切换 **列总计（Column totals）** 包含每列的总计。- 切换 **列最小/最大（Column min/max）** 包含每列中出现的最小值和最大值的摘要。

要生成报告，请配置对话框中的剩余选项；然后点击 **应用（Apply）**（对于 X–Y 报告或场输出报告）或 **确定（OK）**（对于探针值报告）。完成后，点击 **取消（Cancel）** 关闭对话框。
