# 19.12.7 提取平滑网格







优化过程完成后，您可以选择以文件形式提取优化模型表面的平滑等值面网格表示，该文件可以传输到 CAD 系统或返回到 Abaqus/CAE。您可以从拓扑或形状优化中提取平滑的表面网格。

优化的模型通常是四面体和六面体元素的组合，并且可以包括大量未包含在将生成的等值面表示中的内部元素。此外，模型的表面可能会粗糙且不规则，具体取决于单元密度。平滑操作是一个迭代过程，它置换表面的节点并在模型表面生成平滑的三角形网格。

平滑操作之后可以进行数据缩减操作，该操作将相邻三角形与几乎重合的平面组合起来并减少表面三角形的数量。数据缩减不断迭代，直到三角形数量减少达到原始三角形数量的指定百分比。您可以指定确定相邻平面是否重合的角度。

最后，您可以选择应用可选的过滤操作来消除局部不规则之处。滤波在平滑操作之前应用。

**要提取平滑的表面网格：**

1. 从主菜单栏中，选择****优化**![](../graphics/images/arrow.gif)**提取**![](../graphics/images/arrow.gif)***优化进程名称*****。您还可以通过在**优化流程管理器**中选择优化流程名称并单击**提取**来平滑结果。将出现“**提取曲面网格**”对话框。
2. 输入要提取的输出文件的名称。
3. 选择将生成的文件的格式。您可以选择以下任一选项： - 可导入到 Abaqus/CAE 中的 Abaqus 输入文件 (`.inp`)。 - 可由 3D CAD 系统（例如 CATIA V5 和 SolidWorks）读取的 STL 文件 (`.stl`)。
4. 在 **设计周期** 字段中，输入应从中提取等值面的设计周期。默认情况下，“编辑平滑选项”对话框显示最后一个设计周期。
5. 输入一个介于 0 到 1 之间的值，指定 **Iso 值**。平滑过程确定哪些元素位于模型表面上，并使用等值来计算在元素的内边缘上创建新节点的位置。增加等值会导致等值面向模型内部移动，从而导致模型体积减小。对于具有薄组件的结构，不应输入大于`0.7`的等值，以防止结构断开连接。
6. 输入 **缩减百分比**，它定义在数据缩减操作期间应删除的面的百分比。值 0 表示不应删除任何面。值 100 表示在给定缩减角度值的情况下，当无法删除更多面时数据缩减停止。
7. 输入 0 到 90 之间的值，指定 **缩小角度**。缩减角度定义了节点处的相邻面之间的最大角度，使得可以在数据缩减期间移除该节点。
8. 选择迭代平滑周期的数量。较大的值会导致模型更平滑，但会导致计算时间增加。实际上，5 到 10 个平滑周期通常就足够了。进一步平滑可能会导致薄部件变窄或收缩。您可以通过选择`0`值来关闭平滑。
9. 输入一个介于 0 到 1 之间的值，指定 **目标体积**。实体模型的目标或相对体积是等值面定义的体积与原始体积（根据优化生成的实体单元的体积计算得出）的比率。同样，壳模型的目标面积是等值面定义的体积与原始面积（根据优化生成的壳单元的面积计算得出）的比率。如果模型同时包含实体单元和壳单元，则仅考虑实体单元。计算的体积不考虑实体模型内的空隙。如果您指定目标卷，数据缩减过程将计算将产生指定卷的等值（您输入的等值值将被忽略）。如果您为目标卷输入的值过大（大于用等值 0 计算的卷），或者为目标卷输入过小的值（小于用等值 1 计算的卷），则数据缩减过程会生成错误。如果数据缩减过程在 20 次迭代之前未能收敛，也会产生错误。
10. 在某些情况下，优化后的模型将包含小的不规则之处，您可以选择通过可选的过滤操作将其删除。在平滑操作之前应用过滤，可以使材料分布更加均匀。选择“**中等**”应用单个过滤周期，选择“**完全**”应用五个过滤周期。
11. 单击 **确定** 以提取平滑的网格。在 TOSCA_POST 目录中创建指定的输出文件（`.inp`或`.stl`）。对于后续运行，文件名后会附加一个唯一的编号；例如，`*name*_002.inp`。 **注意：**如果以 Abaqus 输入文件的形式提取平滑网格，则可以从主菜单栏中选择****文件**![](../graphics/images/arrow.gif)**导入**![](../graphics/images/arrow.gif)**模型****，将网格导入到 Abaqus/CAE 中。有关详细信息，请参阅["Importing a model," Section 10.8](pt02ch10hla01.md)。
