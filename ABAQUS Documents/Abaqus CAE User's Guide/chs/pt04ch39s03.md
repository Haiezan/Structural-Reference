# 39.3 指定子结构的保留节点自由度和荷载工况







为分析定义一个或多个子结构生成步骤后，必须为子结构定义 **保留节点自由度** 边界条件。子结构节点保留的自由度是外部自由度，可用于分析；指定节点的所有其他自由度均假定为子结构内部的自由度，并且不考虑到分析中。当您将此分析中的子结构导入模型以供子结构使用时，Abaqus/CAE 将这些节点显示为浅蓝色十字，这使您能够从零件实例或装配体中轻松选取它们。
