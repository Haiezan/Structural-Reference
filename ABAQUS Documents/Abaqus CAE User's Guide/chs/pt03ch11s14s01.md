# 11.14.1 定义放样剖面







放样截面表示放样特征在沿放样路径的特定点处具有的形状。创建放样特征至少需要两个截面。您可以创建其他截面来控制起始截面和结束截面之间的放样形状。在实体或切割放样中，每个放样部分必须是没有分支的闭环。在壳放样中，放样部分可以全部打开或全部关闭。您可以定义平面或非平面放样截面来创建放样特征。创建放样后，放样中的截面数量及其顺序就无法更改。

您可以通过从当前视口中零件的现有边拾取来创建放样剖面。可以选择任意边；例如：
- 定义拉伸、旋转或扫掠特征的边线。
- 定义平面线或壳特征的边。
- 花键线特征。

您可以使用多个特征的各个边来定义单个截面。但是，在基准平面上绘制平面线是可用于定义放样截面的最简单方法之一。使用最简单的方法定义放样截面将为您提供更多控制，并产生更强大的放样特征。

您不能直接修改放样剖面。创建放样特征后，您可以使用特征操作工具集来编辑创建放样截面中使用的边的特征。移动放样剖面中使用的顶点或边将更改剖面的形状以及相应放样特征的形状。
