# 17.4.5 限制种子







默认情况下，网格种子仅规定目标网格密度。当您使用自由网格划分技术生成三角形或四面体单元时，Abaqus/CAE 通常会与网格种子完全匹配。然而，在其他情况下，Abaqus/CAE 可能会改变单元分布，以便能够成功生成网格。如果要防止 Abaqus/CAE 更改单元分布，可以通过约束沿该边的种子来固定沿该边的特定数量的单元。您只能约束边缘种子，而不能约束零件或实例种子。

您可以将以下三种状态中的任意一种分配给沿边的一组种子：

**无约束**

这是默认设置。沿边的元素数量可以增加或减少，以便网格可以变得比种子指定的更密或更粗。不受约束的种子显示为空心圆圈。

**部分受限**

沿边的元素数量在网格生成过程中可能会增加，但不能减少。此约束允许网格变得比种子指定的更密，但不会更粗。部分受约束的种子显示为向上的三角形。

**完全约束**

网格生成过程无法更改沿边的约束种子指定的元素数量。当种子受到完全约束时，网格生成将尝试允许节点的位置与种子的位置精确对应。然而，不能保证种子和节点位置之间的精确匹配。完全约束的种子显示为正方形。

Abaqus/CAE 始终在区域的每个几何顶点处创建完全约束的种子，以指示有限元节点将位于每个顶点处。

在许多情况下，网格生成器必须重新分配元素（并偏离种子的数量和位置）才能成功生成网格。为了获得网格划分成功的最大可能性，请让种子不受约束，或者至少避免在给定零件或零件实例中完全约束大量种子，以便网格生成器具有尽可能多的自由度来重新分配种子。
