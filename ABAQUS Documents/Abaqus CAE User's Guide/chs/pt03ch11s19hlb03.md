# 11.19.3 选择新零件的类型







使用 **创建零件** 对话框中的 **类型** 单选按钮选择要在 Abaqus/Standard 或 Abaqus/Explicit 模型中创建的零件的类型。在 Abaqus/CFD 模型中，只能创建流体零件。 Abaqus/CAE 在建模过程中携带零件的类型；例如，您无法将截面和材料属性分配给刚性零件，无法对分析刚性零件进行网格划分，并且只能将欧拉截面和材料分配字段分配给欧拉零件。如果允许，您可以在创建零件后更改零件的类型，方法是在模型树中的零件上单击鼠标按钮 3，然后从出现的菜单中选择 **编辑**。

在 Abaqus/Standard 或 Abaqus/Explicit 模型中，新零件的类型可以设置为以下类型之一：

**可变形**

您可以创建或导入的任何任意形状的轴对称、二维或三维零件都可以指定为可变形零件。可变形零件表示在载荷作用下可以变形的零件；负载可以是机械负载、热负载或电负载。默认情况下，Abaqus/CAE 创建可变形的零件。

**离散刚性**

离散刚性零件与可变形零件相似，因为它可以是任意形状。然而，离散刚性零件被假定为刚性的，并在接触分析中用于对不能变形的实体进行建模。

**分析刚性**

分析刚性零件与离散刚性零件类似，用于表示接触分析中的刚性表面。然而，分析刚性零件的形状不是任意的，必须由一组草绘直线、圆弧和抛物线形成。

**欧拉**

欧拉零件用于定义材料可以在其中流动以进行欧拉分析的域。欧拉零件在分析过程中不会变形；相反，零件内的材料在载荷作用下会变形，并且可以流过刚性单元边界。有关欧拉分析的更多信息，请参阅[Chapter 28, "Eulerian analyses](pt04ch28.md)。”

创建离散刚性零件或分析刚性零件后，还必须执行以下操作：
- 指定刚体参考点。您可以在“加载”模块中将约束或指定运动应用于刚体参考点，并且相同的约束或运动将应用于整个刚性零件。有关详细信息，请参阅["The reference point," Section 11.8.1](pt03ch11s08s01.md)。
- 如果零件是离散刚性零件或分析刚性零件，则必须使用装配模块中的曲面工具集来选择零件的哪一侧代表外表面。有关详细信息，请参阅[Chapter 73, "The Set and Surface toolsets](pt06ch73.md)。”

**选择新零件的类型：**

1. 从 **创建零件** 对话框的中间，选择所需的 **类型** 单选按钮。
2. 选择完选项后，单击“**继续**”关闭“**创建零件**”对话框。草绘器启动，您可以绘制新零件基本特征的轮廓。
