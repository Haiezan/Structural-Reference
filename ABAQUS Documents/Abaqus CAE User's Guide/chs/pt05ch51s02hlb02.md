# 51.2.2 探测节点







您可以使用查询工具集获取节点数据以及所选模型中的属性值或所选模型图的分析结果。当您将光标置于模型中或当前视口中模型图中的节点上时，Abaqus/CAE 会突出显示该节点。您可以选择在数据表中显示有关该节点的以下信息：
- 节点的标签，
- 节点的原始坐标，
- 节点的变形坐标（使用当前变形场输出变量），
- 共享节点的元素，以及
- 当前场输出变量（如果当前场输出变量是张量或向量，则包括分量值）。如果在当前视口中选择模型，则当前场输出变量反映模型属性的值，例如所选节点处的载荷。

**注意：**您可以显示或隐藏表中的任何列，方法是在表中单击鼠标按钮 3，选择 **编辑可见列**，然后在出现的对话框中切换任何列。

您还可以通过直接输入节点标签来获取节点的数据，并且可以获取所选显示组中包含的所有节点的数据。这些方法使您能够探测节点数据，而无需将光标定位在模型图中的位置上。

将节点添加到数据表时，可以切换该行的复选框以在视口中显示该节点的探测注释，也可以切换标题行中的复选框以显示所有选定节点的探测注释。探针注释显示节点的标签以及当前字段输出变量在节点处的结果。您可以自定义注释文本的字体和颜色。

为了帮助在视口中定位节点，您可能需要在使用查询工具集之前生成节点符号可见的模型图。有关详细信息，请参阅["Customizing node symbols," Section 55.5.5](pt05ch55s05hlb05.md)。

**探测节点：**

1. 找到**探测值**对话框。在可视化模块工具箱中，单击![](../graphics/ico_probe.png)。将出现 **探针值** 对话框。
2. 选择控制 Abaqus/CAE 将显示的探测数据和结果的步骤、框架、场输出变量和结果选项。如果您正在探测当前模型数据库中的数据，则框架选择不可用，并且您选择的字段输出变量是当前视口中显示的模型的属性。在 **Probe Values** 对话框中，**Field Output** 字段标识当前步骤、帧和字段输出变量。 - 要更改步骤或帧，请单击![](../graphics/ico_stepSmall.png)。 - 要更改现场输出变量，请单击![](../graphics/ico_fieldOutput.png)。 - 要更改结果选项，请从主菜单中选择****结果**![](../graphics/images/arrow.gif)**选项****。
3. 在对话框的 **探测值** 部分中，执行以下操作： 1. 单击 **探测** 箭头，然后选择 **节点**。 2. 如果当前主输出变量是张量或向量变量，请单击 **Components** 箭头并执行以下操作之一： - 对于张量变量，选择 **Selected** 以仅显示变量的选定分量，选择 **All Direct** 以在查询中包含张量变量的所有直接分量，或选择 **All Primary** 以在查询中包含张量变量的所有主不变量。 - 对于向量变量，选择 **Selected** 仅显示变量的选定分量，或选择 **All** 以在查询中包含向量张量变量的所有分量。可用的节点信息类型显示在数据表的列中。
4. 在数据表中的任意位置单击鼠标按钮 3，从出现的列表中选择“编辑可见列”，然后切换要在数据表中显示为列的节点信息类型。当打开所需的信息类型时，Abaqus/CAE 会将该类型的数据显示为数据表中的列。但是，所有列（无论是否可见）都包含在探测值输出文件中。 **注意：**仅当当前主要变量是基于元素的变量（例如应力或应变）时，现场输出才可用。
5. 要指定要探测的节点，请选择 **从视口中选择**、**键入标签** 或 **选择显示组**，并使用以下技术之一： **直接从视口中选择节点：** 1. 移动光标以探测当前视口中的模型图。当您将光标置于节点上时，Abaqus/CAE 会在数据表的第一行中显示您请求的项目的值。 2. 要选择并累积感兴趣的值，并允许随后将这些值写入文件，请单击鼠标按钮 1，同时将光标放在节点上。 Abaqus/CAE 将数据表第一行的数据存储到数据表的存储行中。 **直接将节点标签输入到表中：** 1. 单击 **零件实例** 箭头，然后选择要从中选择节点标签的零件实例。 2. 输入要包含的一个或多个节点标签。您可以同时指定多个节点标签，并用逗号分隔它们。 **选择显示组中包含的节点：** 1. 单击 **显示组** 箭头，然后选择包含要包含在查询中的节点的显示组。如果更改显示组的选择，Abaqus/CAE 会将数据表的内容替换为新选择的显示组中的节点。 2. 如果需要，可以通过从视口中选择其他节点或通过节点标签键入它们来将其他节点附加到列表中。
6. 重复节点选择步骤，直到 **Probe Values** 对话框中的数据表包含所有要查询的节点。
7. 要从数据表中删除数据或者清除或排序表，请单击鼠标按钮 3。有关详细信息，请参阅["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md)。
8. 要使用数据表中的节点创建显示组，请执行以下操作： 1. 单击数据表中的鼠标按钮 3，然后从出现的菜单中选择 **创建显示组**。将出现 **创建显示组** 对话框。 2. 输入新显示组的名称，然后单击“**确定**”。
9. 要显示数据表中任何节点的探测注释，请选中与要注释的节点对应的行的复选框，或打开标题行中的复选框以显示所有选定节点的探测注释。 **注意：**您可以在“通用绘图选项”对话框中自定义节点探针注释的颜色和字体。有关详细信息，请参阅["Overview of common plot options," Section 43.3](pt05ch43hla01.md)。
10. 要保存您选择的数据，请单击“**写入文件**”。有关详细信息，请参阅[Chapter 54, "Generating tabular data reports](pt05ch54.md)。”
11. 要退出探测模式，请执行以下操作之一： - 单击提示区域中的取消![](../graphics/afxI_cancel.png)按钮。 - 单击“**探测值**”对话框中的“**取消**”。 - 单击鼠标按钮 2。 - 以其他模式（例如未变形、变形）生成绘图。
