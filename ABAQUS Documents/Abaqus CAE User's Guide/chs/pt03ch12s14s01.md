# 12.14.1 定义复合铺层时使用铺层表







创建复合叠层（对于壳、连续体壳或实体）时，可以使用层表中的一行来输入有关叠层中层的信息。行数决定铺层中的层数，层表的第一行对应于铺层底部的层数。层表的每一行都要求以下信息：
- 层是否被激活。
- 层的名称。
- 层所分配到的区域。
- 该层中使用的材料的名称。
- 层厚度。
- 指定层的参考方向的坐标系。
- 相对于参考方向的旋转角度。
- 如果您要为分析期间集成的复合材料铺层指定属性，则层中集成点的数量。

有关在 Abaqus/CAE 中使用表的一般信息，请参阅["Entering tabular data," Section 3.2.7](pt01ch03s02s07.md)。

当您在表中输入层数据时，层区域会突出显示，并且层方向会显示在视口中的模型上。您可以修改显示以关闭突出显示、显示铺层方向或选择要显示的方向向量。

如果 Abaqus/CAE 尝试在系统奇点处（即，用户选择的坐标系和来自几何体或元素的几何法线无法解析为用于 Abaqus/CAE 中显示目的的有效方向的点）绘制复合铺层中层方向的坐标系，则坐标系将被折叠绘制。

如果使用离散字段指定铺层方向，则铺层方向将相对于零件的基础坐标系显示，并且不会显示铺层方向。对于连续体壳单元，Abaqus/CAE 不会将显示的方向投影到中平面表面上。在这两种情况下，您都可以执行数据检查并在可视化模块中查看输出数据库以验证方向。有关详细信息，请参阅["Performing a data check on a model," Section 19.7.3](pt03ch19s07hlb03.md)。

有许多工具可用于在层表中添加或删除层。大多数工具在层表上方显示为图标。某些工具只能通过单击层表上的鼠标按钮 3 并从出现的上下文菜单中选择一个项目来访问。可以使用以下工具：

**编辑**

编辑所选表格单元格的内容。例如，您可以从视口中的零件中选择一个区域、选择材料或输入厚度。在大多数情况下，您可以选择多个单元格，Abaqus/CAE 会将您的更改应用到每个选定的单元格。您可以选择列标题来选择整个列。

**编辑**工具可从上下文菜单中使用。或者，您可以双击 **Region**、**Material** 或 **CSYS** 列中的单元格来编辑该单元格。您还可以双击列标题，将相同的编辑应用于该列中的所有单元格。

**向上或向下移动层**

将选定的层在层表中向上或向下移动一行。单击![](../graphics/ico_tableMoveRowUp.png)将选定的层向上移动一行，或单击![](../graphics/ico_tableMoveRowDown.png)将选定的层向下移动一行。

**复制之前或之后的层数**

通过在选定层之前或之后复制选定层来创建新层。单击![](../graphics/ico_tableCopyRowBefore.png)将所选层复制到所选层之前，或单击![](../graphics/ico_tableCopyRowAfter.png)将所选层复制到所选层之后。

**在之前或之后插入行**

在每个选定层之前或之后插入一个空行。单击![](../graphics/ico_tableInsertRowBefore.png)在选定层之前插入空行，或单击![](../graphics/ico_tableInsertRowAfter.png)在选定层之后插入空行。

**删除层**

删除选定的层。单击![](../graphics/ico_delete.png)删除选定的层。

**图案层数**

使用图案复制选定的层。单击![](../graphics/ico_pliesPattern.png)对选定的层进行图案化。

图案层复制到的位置可以是铺层中的第一层或最后一层，也可以是所选层中的第一层或最后一层。您还可以通过选择下面要复制的层的名称来指定复制位置。如果选择要复制的层的名称，则可以选择将复制的层插入到层表中或覆盖任何现有层。

该模式可以是以下之一：

**N 副本**

选定层的指定数量的副本。

**对称**

所选层的单个副本。复制的层会重新排序，以便原始层的顺序和复制的层的顺序关于指定位置对称。

**反转层**

反转两个或多个选定相邻层的顺序。单击![](../graphics/ico_pliesInvert.png)可反转所选层的顺序。

**从文件读取**

从 ASCII 文件读取层数据。数据可以用空格、制表符或逗号分隔。您可以指定 Abaqus/CAE 应开始从层表的哪一行和哪列读取数据。表的第一列是抑制的状态（真或假）。

**从文件读取**工具只能从上下文菜单中使用。

**注意：**文件的 **Region** 列中的数据是指您零件中的命名集。区域名称区分大小写。因此，您在文件中使用的区域名称必须与零件中现有集的名称相同。对于 **CSYS** 列中的数据也是如此 - 您在文件中使用的坐标系的名称必须与零件中现有基准坐标系的名称相同。

**写入文件**

将整个层表写入指定文件。 Abaqus/CAE 使用逗号来分隔数据。

**写入文件**工具只能从上下文菜单中使用。
