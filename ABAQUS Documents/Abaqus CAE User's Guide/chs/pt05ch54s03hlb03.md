# 54.3.3 选择场输出截面点





如果您的场输出报告包含截面点变量，您可以选择 Abaqus/CAE 报告这些变量值的位置。默认是在所有可用截面点报告值。

**选择场输出截面点：**

1. 找到截面点选项。从主菜单栏中，选择 ****报告（Report）**![](../graphics/images/arrow.gif)**场输出（Field Output）****；然后在出现的对话框中点击 **变量（Variables）** 选项卡。
2. [使用复选框方法](pt05ch54s03hlb02.md) 选择截面变量。
3. 在对话框底部的 **截面点（Section point）** 标签处，执行以下操作之一：- 点击 **所有（All）** 在所有可用截面点报告值，或- 点击 **单个（Single）** 表示您将选择一个截面点来报告值。如果您已选择至少一个单元截面点变量，**设置（Settings）** 按钮将变为可用。
4. 要选择一个或多个截面点，请点击 **设置（Settings）**。出现 **场报告截面点设置（Field Report Section Point Settings）** 对话框。使用此对话框选择所需的截面点。具体信息请参阅 ["选择截面点数据，" 第 42.5.9 节"](pt05ch42s03s01.md#usv-res-visvariableshelldb) 中的"按类别选择截面点数据"。
5. 点击 **确定（OK）** 应用您的截面点选择并关闭 **场报告截面点设置（Field Report Section Point Settings）** 对话框。

要生成报告，请在 **设置（Setup）** 页面上配置剩余选项，**报告场输出（Report Field Output）** 对话框中也有此页面；然后点击 **应用（Apply）**。完成后，点击 **取消（Cancel）** 关闭对话框。
