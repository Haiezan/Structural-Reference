# 15.13.21 定义空腔​​辐射相互作用







本节介绍如何在 Abaqus/CAE 中创建空腔辐射相互作用并使用空腔对称性来减少计算费用。涵盖以下主题：
-["Defining cavity radiation properties and view factors](pt03ch15s13s03.md#usi-itn-help-cavityraddef)”
-["Defining cavity radiation symmetry](pt03ch15s13s03.md#usi-itn-help-cavradsymm)”

### 定义腔辐射特性和视角因子

您可以通过创建空腔辐射相互作用来模拟外壳中辐射引起的传热。从主菜单栏中选择****交互**![](../graphics/images/arrow.gif)**创建****，然后选择曲面。有关空腔辐射的简要概述，请参阅["Understanding interactions," Section 15.3](pt03ch15s03.md)。有关更详细的讨论，请参阅["Cavity radiation," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-acavityradiation)。

**定义空腔辐射相互作用：**

1. 从主菜单栏中，选择****交互**![](../graphics/images/arrow.gif)**创建****。 **提示：**您还可以使用相互作用模块工具箱中的![](../graphics/ico_interaction.png)工具创建腔辐射交互。
2. 在出现的 **创建交互** 对话框中，执行以下操作： - 为交互命名。有关命名对象的更多信息，请参阅["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md)。 - 选择步骤。您只能在传热或热电耦合步骤期间定义腔辐射。 - 选择**空腔辐射**交互类型。
3. 单击“**继续**”关闭“**创建交互**”对话框。
4. 使用以下方法之一选择型腔表面。仅选择应用[cavity radiation interaction property](pt03ch15s14hlb02.md)的曲面部分。 - 使用现有曲面来定义区域。在提示区域右侧，单击“**曲面**”。从出现的 **区域选择** 对话框中选择现有曲面，然后单击 **继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要恢复到其他方法，请单击提示区域右侧的 **在视口中选择** 或 **曲面**。 - 使用鼠标在视口中选择一个区域。 （有关详细信息，请参见["Selecting objects within the current viewport," Section 6.2](pt01ch06s02.md)。）如果模型包含网格和几何体的组合，请从提示区域单击以下选项之一： - 如果要从几何区域选择曲面或顶点，请单击“**几何体**”。 - 如果要从原生或孤立网格选择中选择曲面或节点，请单击 **网格**。您可以使用角度方法从几何体中选择一组面或边，或者从网格中选择一组单元面。有关更多信息，请参阅["Using the angle and feature edge method to select multiple objects," Section 6.2.3](pt01ch06s02hlb03.md)。 Abaqus/CAE 显示 **编辑交互** 对话框，并且表面名称或`(Picked)`出现在 **属性** 选项卡页面上的 **表面** 列中。
5. 要定义其他型腔表面，请在表中单击鼠标按钮 3，选择 **添加行**，然后执行以下操作之一： - 双击 **表面** 列中的空单元格。 - 在空表行中单击鼠标按钮 3，然后选择 **编辑曲面**。 **注意：**您还可以使用这些技术来替换现有的型腔表面；当您编辑现有曲面时，Abaqus/CAE 不会指示或保留您的原始选择。
6. 在 **定义** 字段中选择型腔类型。 - 选择 **Closed** 指定一组用于辐射的封闭表面。 - 选择“**开放**”以包含对周围环境的一些辐射，并为开放腔定义指定**环境温度**。
7. 指定**属性**选项。 1. 选择**阻挡表面检查**。默认情况下，Abaqus 在执行辐射视图因子计算时会检查腔内的阻挡表面。 - 选择 **全部** 表示所有阻止检查均处于活动状态。 - 选择 **无** 以跳过阻止检查。 - 选择 **Partial** 以指定 Abaqus 应检查的潜在阻挡面。有关详细信息，请参阅["Controlling checks for surface blocking" in "Cavity radiation," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-acavityradiation-viewfactor-blocking)。 2. 如果选择 **Partial** 选项，请双击 **Blocking Surface** 表中的空单元格以选择您希望 Abaqus 检查的表面。 **提示：**您还可以在**阻挡表面**表中单击鼠标按钮 3，然后选择**编辑表面**、**添加行**或**删除行**来编辑该表。表面选择方法与步骤4中所述相同。 3. 选择热反射行为。 - 对于灰体辐射，选择**是**。灰体的发射率在 0 到 1 之间，由空腔辐射相互作用特性定义。 - 对于黑体辐射选择**否**。黑体的发射率固定为 1——不反射热量。 4. 如果您在上一步中选择了灰体辐射，则必须为表中的每一行指定空腔辐射相互作用属性。您可以使用以下任意方法： - 单击“属性”列中的单元格以选择预定义的腔辐射相互作用属性。 - 单击表中的鼠标按钮 3，然后选择 **创建属性** 以创建新的交互属性，或选择 **编辑属性** 以编辑所选行中现有的腔辐射交互属性。 （有关详细信息，请参阅["Defining a cavity radiation interaction property," Section 15.14.3](pt03ch15s14hlb02.md)。） - 在 **发射率** 列中输入一个值。 Abaqus/CAE 自动创建具有默认名称和指定发射率的腔辐射相互作用属性。完成后，表中的每一行都包含一个表面和一个发射率值或`(table)`，表明交互属性中定义了表格发射率。如果您输入发射率值，**属性**单元格可能为空——当您关闭 **编辑交互** 对话框时，Abaqus/CAE 将添加默认交互属性名称。
8. 指定 **查看因素** 选项。 1. 如果需要，打开 **指定遮挡范围** 并输入一个距离值，超出该距离，Abaqus 不应由于遮挡效应而计算视图系数。 2. 指定**精度公差**。默认值为 0.05。视角系数公差表示与理想视角系数之和的允许偏差。如果超出封闭腔的公差，Abaqus 就会结束分析。如果超出开放腔体的容差，就会对周围环境产生辐射。 3. 指定**无穷小面面积比**。默认值为 64。该值表示最大面面积与最小面面积的比率。 Abaqus 计算每个面对的该比率。 4. 指定**每条边的高斯积分点**。默认值为 3。该值用于数值积分方法。可能的值范围为 1 到 5。 5. 指定 **集总面积距离平方值**。默认值为 5。该值表示每个面对质心之间距离的平方与较大面的面积之比。如果计算值大于此设置，Abaqus 将使用集总面积近似进行积分。 6. 如果需要，单击 **默认** 按钮将所有视图系数条目重置为 Abaqus 默认值。如果一对小平面的计算值小于或等于 **集总面积距离平方值** 设置并且超过 **无穷小小平面面积比率**，则 Abaqus 将使用无穷小到有限面积近似。如果超过 **集总面积距离平方值** 但未超过小平面面积比，Abaqus 会完成轮廓积分的数值积分以获得准确值。有关详细信息，请参阅["Controlling the accuracy of view factor calculations" in "Cavity radiation," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-acavityradiation-viewfactor-vtol)。
9. 指定**对称**选项。有关可用腔辐射对称类型的详细信息，请参阅["Defining cavity radiation symmetry](pt03ch15s13s03.md#usi-itn-help-cavradsymm)。”
10. 在 **编辑模型属性** 对话框中指定正在使用的温标上的绝对零温度![](../graphics/usi_eqn00574.gif)和 Stefan-Boltzmann 常数![](../graphics/usi_eqn00416.gif)，如["Specifying model attributes," Section 9.8.4](pt02ch09s07hlb04.md)中所述。
11. 单击 **确定** 创建交互并关闭编辑器。

### 定义腔辐射对称性

使用对称性可以减少腔模型的计算量。可用的对称性和组合因模型类型而异。["Combining symmetries" in "Cavity radiation," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-cavityrad-cavsymmcombine)中的表指示可用的对称组合。您可以使用以下对称类型来完成型腔定义：

****反射****

选择 **反射对称数**，并为每次反射选择参考 *z* 对称值（轴对称模型）、对称轴（二维模型）或对称平面（三维模型）。

Abaqus/CAE 将反射表面添加到腔定义中，并减少模型中允许的剩余对称数量。切换**突出显示**可在视口中查看当前参数选择。

****定期****

选择**周期性对称性的数量**以及每个周期性对称性的重复次数。对于轴对称模型，选择参考 *z* 对称值和周期性 *z* 距离值。对于二维和三维模型，为每个周期对称性分别选择对称轴和距离向量或对称平面和距离向量。

Abaqus/CAE 将周期曲面添加到您为 **属性** 选项选择的型腔曲面，并减少模型中允许的剩余对称数。切换**突出显示**可在视口中查看当前参数选择。

****循环****

开启**使用循环对称**并选择扇区总数。选择对称点和对称轴上的点（二维模型）或对称轴上的第一点和第二点以及对称平面上的点（三维模型）。

循环对称通过绕对称轴顺时针旋转原始几何体来创建新的扇形。循环对称不适用于轴对称模型。

必须满足以下条件：
- 对于二维模型，对称轴上的选定点必须位于定义原始扇区的几何图形的顺时针一侧。
- 对于三维模型，对称平面上的选定点必须位于定义原始扇区的几何图形的逆时针一侧。
- 扇区总数必须定义一个完整的圆 (360)。如果更改扇区数，则必须重新定义几何图形以表示模型的正确部分。

有关循环对称性的更多信息，包括显示扇区定义的图，请参阅["Cyclic symmetry" in "Cavity radiation," Section 41.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-cni-cavityrad-cavsymmcyclic)。

**警告：**Abaqus/CAE 不会检查定义的对称性是否会生成物理上真实的空腔模型。
