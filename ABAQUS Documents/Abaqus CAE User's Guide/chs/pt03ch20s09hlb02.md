# 20.9.2 打开或关闭捕捉







当您想要选择草绘器网格点或将对象拖动到网格上的特定点时，捕捉到网格可帮助您定位光标。启用捕捉并且将光标移动到网格点附近时，辅助光标会跳转或捕捉到该网格点。在“草绘器选项”对话框中切换“捕捉到网格”以启用或禁用捕捉。 （默认情况下，启动草绘器时会启用捕捉。）您选择的捕捉行为适用于 Abaqus/CAE 会话中的所有草图。如果预选点和网格点都靠近光标位置，则草绘器预选将覆盖捕捉。

**要打开或关闭捕捉：**

1. 从草绘器工具箱底部，选择草绘器自定义工具![](../graphics/ico_sketchOptions-nls.png)。将出现 **草绘器选项** 对话框。
2. 在 **常规** 选项卡页面中，切换 **对齐网格**。当**对齐网格**启用时，当您在草图周围移动时，辅助光标会对齐到附近的网格点。根据网格大小、视口放大率以及预选是否处于活动状态，您还可以选择除网格点和现有草图几何体之外的其他点。当**对齐网格**关闭时，当您在草图周围移动时，辅助光标将与主光标对齐。您可以选择工作表上的任意位置。
3. 选择所需的自定义选项后，单击“**确定**”应用更改并关闭“**草绘器选项**”对话框。
