# 11.19.1 使用“创建零件”对话框定义零件的属性







### 11.19.1 使用 **创建零件** 对话框定义零件的属性

创建零件时，首先使用“创建零件”对话框定义零件的属性，然后使用“草图”绘制基础特征的二维轮廓。您可以使用 **创建零件** 对话框来定义以下内容：

****姓名****

使用 **创建零件** 对话框顶部的 **名称** 文本字段来命名您正在创建的零件。要重命名零件，请从主菜单栏中选择****零件**![](../graphics/images/arrow.gif)**重命名****。有关有效名称的信息，请参阅["Using basic dialog box components," Section 3.2.1](pt01ch03s02s01.md)。

创建零件后，Abaqus/CAE 会在当前视口的标题栏中显示新零件的名称。

****建模空间****

使用**建模空间**单选按钮选择新零件的建模空间。您可以在 Abaqus/Standard 或 Abaqus/Explicit 模型中将零件定义为三维、二维（平面）或轴对称零件。在 Abaqus/CFD 模型中只能创建三维零件。

如果您创建轴对称可变形零件，则可以在“创建零件”对话框中打开“包括扭曲”，以在模型中包含扭曲自由度。创建零件后，您可以通过在模型树中的零件上单击鼠标按钮 3 并从出现的菜单中选择 **编辑** 来更改零件的建模空间。有关详细信息，请参阅["Choosing the modeling space of a new part," Section 11.19.2](pt03ch11s19hlb02.md)。

****类型****

在 Abaqus/Standard 或 Abaqus/Explicit 模型中，使用 **类型** 单选按钮选择新零件的类型。您可以将零件定义为可变形零件、离散刚性零件、分析刚性零件或欧拉零件。在 Abaqus/CFD 模型中只能创建流体零件。如果允许，您可以在创建零件后更改零件的类型，方法是在模型树中的零件上单击鼠标按钮 3，然后从出现的菜单中选择 **编辑**。  有关详细信息，请参阅["Choosing the type of a new part," Section 11.19.3](pt03ch11s19hlb03.md)。

****基本功能****

使用 **基本特征** 字段定义新零件基本特征的形状和类型。 Abaqus/CAE 显示的形状和类型选项取决于零件的建模空间和类型。创建零件后，您无法更改其基本特征的类型。有关详细信息，请参阅["Choosing the base feature of a new part," Section 11.19.4](pt03ch11s19hlb04.md)。

****大致尺寸****

使用近似尺寸文本字段输入零件的尺寸。 Abaqus/CAE 使用您输入的尺寸来计算草绘器图纸的尺寸及其网格的间距。有关详细信息，请参阅["Setting the approximate size of the new part," Section 11.19.5](pt03ch11s19hlb05.md)。创建零件并开始绘制其轮廓草图后，您可以使用 **草图** 自定义选项来增加图纸尺寸。要显示草绘器自定义选项，请单击草绘器工具箱底部的![](../graphics/ico_sketchOptions-nls.png)工具。
