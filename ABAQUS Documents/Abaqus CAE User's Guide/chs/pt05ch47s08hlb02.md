# 47.9.2 自定义 X–Y 绘图图例的标题

### 47.9.2 自定义 X–Y 绘图图例的标题






使用 **标题（Title）** 选项来显示或隐藏 X–Y 绘图图例的标题，并自定义其内容。

**要自定义 X–Y 绘图的图例标题：**

1. 找到 **文本（Text）** 选项。选择 ****选项（Options）**![](../graphics/images/arrow.gif)**XY 选项（XY Options）**![](../graphics/images/arrow.gif)**图表图例（Chart Legend）****；或点击 ![](../graphics/ico_xyChartLegendOptions.png)，该图标位于可视化模块工具箱的 X–Y 绘图工具中。**提示：**当显示 X–Y 绘图时，您还可以通过双击视口中的图例直接打开此对话框。当活动过程生效时，您无法直接从视口中选择 X–Y 绘图中的组件；在从 X–Y 绘图中选择项目之前，请取消当前过程。图例的 **文本（Text）** 选项出现在 **内容（Contents）** 页面的下半部分。
2. 从 **图表（Charts）** 字段中，选择一个或多个要自定义其图例的 X–Y 绘图。**注意：**要使 X–Y 曲线可供选择，您必须首先绘制它。
3. 如果 **内容（Contents）** 选项卡尚未选择，请点击它。
4. 如果需要，在 **标题（Title）** 字段中输入 X–Y 绘图图例的标题。
5. 选择 X–Y 绘图图例标题的字体。1. 点击 ![](../graphics/ico_text16.png)。Abaqus/CAE 显示 **选择字体（Select Font）** 对话框。2. 使用 **选择字体（Select Font）** 对话框中的方法选择新字体、大小和样式。有关更多信息，请参阅 ["自定义字体，" 第 3.2.8 节"](pt01ch03s02s08.md)。3. 点击 **应用（Apply）** 查看字体选择的效果。4. 点击 **确定（OK）** 关闭 **选择字体（Select Font）** 对话框。X–Y 绘图图例标题变为所选的字体、大小和样式。
6. 选择 X–Y 绘图图例标题的颜色。1. 点击颜色样本 ![](../graphics/usi-colorsample.png)。Abaqus/CAE 显示 **选择颜色（Select Color）** 对话框。2. 使用 **选择颜色（Select Color）** 对话框中的方法选择新颜色。有关更多信息，请参阅 ["自定义颜色，" 第 3.2.9 节"](pt01ch03s02s09.md)。3. 点击 **确定（OK）** 关闭 **选择颜色（Select Color）** 对话框。X–Y 绘图图例标题文本变为所选颜色。
7. 点击 **关闭（Dismiss）** 关闭 **图表图例选项（Chart Legend Options）** 对话框。
