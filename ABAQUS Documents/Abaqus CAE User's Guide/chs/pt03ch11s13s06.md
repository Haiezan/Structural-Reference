# 11.13.6 控制旋转特征的方向







当您创建具有旋转基础特征的零件或将旋转特征添加到三维零件时，您可以控制旋转方向。如果在旋转特征中包含俯仰，您还可以控制平移方向。以下描述提供了控制旋转方向以及您在 Abaqus/CAE 中创建的任何旋转特征的平移方向（如果适用）的详细信息：

**创建旋转特征时选择方向**

当您完成草图以创建旋转特征时，Abaqus/CAE 将显示草图轮廓，其中包括指示旋转方向的箭头。如果您正在创建具有旋转基础特征的零件，则还会显示旋转轴。此外，Abaqus/CAE 还会显示 **编辑革命** 对话框。

如果需要，旋转视图，直到您可以辨别出指示旋转方向的箭头方向。您可以通过在对话框中单击 **旋转方向** 的![](../graphics/ico_flipArrow.png)来反转旋转方向。视口中的箭头改变方向以显示新的旋转方向。

如果为旋转特征选择“**包括平移**”，视口中会出现第二个箭头，指示沿旋转轴的平移方向。在对话框中单击“**俯仰方向**”![](../graphics/ico_flipArrow.png)可反转平移方向。与旋转方向的更改类似，视口中相应的箭头会更改方向以显示新的平移方向。

**编辑现有旋转零件或特征的方向**

当您选择要编辑的旋转特征时，Abaqus/CAE 会在视口中突出显示所选特征，并出现 **编辑特征** 对话框。

要反转旋转方向，请在 **编辑特征** 对话框中切换 **翻转旋转方向**。单击 **应用** 查看您的更改。

如果您正在编辑的旋转特征包括俯仰，您可以编辑平移方向。切换**翻转俯仰方向**以反转平移方向。单击 **应用** 查看您的更改。

单击 **确定** 接受更改。
