# 75.4 如果我可以使用虚拟拓扑，为什么还要修复零件？







引入虚拟拓扑是创建干净、结构良好的网格的便捷方法。重点是忽略小的几何细节，这些细节会导致分析结果不重要的区域中的元素扭曲。虚拟拓扑放宽了网格符合每个边或顶点的需要，并允许您获得细节较少的更简单的模型。但是，虚拟拓扑不会更改零件实例的基础几何形状。

当您组合两个面形成虚拟面时，Abaqus/CAE 会更新显示以反映忽略的边。然而，为了形成新虚拟面的基础，Abaqus/CAE 保留了组合面的基础几何形状。网格中的节点将放置在底层几何体上。因此，粗网格将在虚拟面上进行插值，而细网格将趋于收敛到底层组合面的几何形状。虚拟拓扑允许通过网格划分参数（例如种子大小）来控制网格定义，并且有助于使网格划分免受几何体（例如小面）施加的约束。

相比之下，您可以使用几何编辑工具集来纠正从第三方应用程序导入零件时通常会出现的小缺陷。您可以使用几何编辑工具集来校正基础几何；例如，缝合小间隙、删除自相交面并重新创建有效的替换面。您还可以使用几何编辑工具集删除冗余拓扑，例如多余的顶点和边。几何编辑工具集修改实际几何，几何限制限制了您应用修复工具的方式；例如，仅当顶点位于两条边共有的曲线上时，才可以删除冗余顶点。

导出虚拟零件实例时，虚拟拓扑会丢失，因为虚拟面、边或顶点不是真实几何体。例如，虚拟面部是对组合面部的底层集合的抽象引用。因此，如果您需要从零件中删除小细节并希望导出保留更改的零件，则应使用几何编辑工具集。
