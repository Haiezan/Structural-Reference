# 59.6 通过投影点创建附着线







定位连接线分为三个步骤：

1. 选择要投影的点。
2. 沿指定方向将点投影到源面上。
3. 从投影点与源面的交点，沿源面的法线将连接线投影到目标面指定的距离或指定数量的目标面。

您可以使用连接线来定义离散紧固件的位置。有关详细信息，请参阅["About fasteners," Section 29.1](pt04ch29s01.md)。

**通过投影点创建附着线：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**附件**![](../graphics/images/arrow.gif)**投影点的线****。 **提示：**您还可以使用![](../graphics/ico_attLineByProj.png)工具通过投影点来创建附着线，该工具位于相互作用模块工具箱中的附着工具中。有关工具箱中附件工具的图表，请参见["Understanding attachment points and lines," Section 59.1](pt06ch59s01.md)。
2. 从当前视口中，选择要投影的点，然后单击提示区域中的“**完成**”。您可以选择顶点、参考点或附着点。
3. 从当前视口中，选择点将投影到的源面，然后单击提示区域中的 **完成**。附着线从方向矢量与源曲面相交的位置开始。
4. 从当前视口中，选择将与连接线相交的目标面，然后单击提示区域中的“**完成**”。附着线从垂直于源面的起点绘制，并在最远的目标面处结束。 Abaqus/CAE 显示 **通过投影点创建附着线** 对话框。如果需要，您可以使用该对话框来编辑点以及源面和目标面。
5. 从对话框中，选择投影点的方法： - 选择“**邻近**”以允许 Abaqus/CAE 绘制从每个点到所选面上最近点的矢量。 - 选择 **方向** 定义沿其投影点的矢量。单击![](../graphics/ico_selectBlue.png)选择矢量的 **起点** 和 **终点**。
6. 要更改目标面，请显示 **目标** 选项卡页，然后执行以下操作： 1. Abaqus/CAE 显示一个箭头，指示源面上的点投影到目标面的方向。单击![](../graphics/ico_flipArrow.png)可反转箭头方向。 2. 选择确定投影矢量长度的方法： - 选择**沿方向的最大投影**，然后输入与连接线相交的目标面的最大数量。 - 选择**投影矢量的最大长度**，然后输入连接线的最大长度。
7. 默认情况下，Abaqus/CAE 创建一个包含连接线端点的集合。如果需要，您可以修改集名称。如果您不想创建包含端点的集，请关闭**创建带有名称的集**。
8. 单击 **确定** 创建连接线。 Abaqus/CAE 显示附着线。有关编辑连接线的信息，请参阅["Editing attachment lines created by projecting points," Section 59.7.3](pt06ch59s03hlb03.md)。
