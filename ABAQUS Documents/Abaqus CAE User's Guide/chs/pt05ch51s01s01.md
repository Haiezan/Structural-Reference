# 51.1.1 模型或模型图的探测







如果当前视口包含模型图（未变形、变形、轮廓、符号或材料方向图）或当前模型数据库中的模型，Abaqus/CAE 允许您探测所选模型或输出数据库的结果。 **探测值** 对话框标识 Abaqus/CAE 将为其获取值的步骤、帧和场输出变量。要了解如何更改步、帧或场输出变量，请参阅["Selecting the results step and frame," Section 42.3](pt05ch42s01.md)和["Selecting the primary field output variable," Section 42.5.3](pt05ch42s03hlb03.md)。

当您探测三维壳或复合实体的模型图时，Abaqus/CAE 返回的结果取决于您在 **剖面点** 对话框中选择的活动位置。当 **Top** 为活动位置时，Abaqus/CAE 从 **Top Location** 剖面点返回结果；当活动位置设置为 **Bottom** 或 **Top and Bottom** 时，Abaqus/CAE 从 **Bottom Location** 剖面点返回结果。有关截面点的更多信息，请参见["Selecting section point data," Section 42.5.9](pt05ch42s03s01.md)。

对于模型图，您可以选择获取基于节点或基于元素的数据和结果，如下所示：

**探测节点**

当您选择探测节点并将光标定位在当前视口中的节点位置上时，Abaqus/CAE 会突出显示该节点并在数据表中显示有关该节点的以下信息： 
- 节点的标签，
- 节点的原始坐标，
- 节点的变形坐标（使用当前变形场输出变量），
- 共享节点的元素，以及
- 当前场输出变量（如果当前场输出变量是张量或向量，则包括分量值）。如果在当前视口中选择模型，则当前场输出变量反映模型属性的值，例如所选节点处的载荷。

无论当前主变量最初是在节点、质心还是积分点位置保存到输出数据库中，您都可以在节点处显示字段输出。 Abaqus/CAE 通过外推节点并根据您选择的选项求平均值来计算质心和积分点数据的节点值。如果节点引用未平均的元素数据，则将显示与探测节点关联的所有元素数据。 For more information on averaging, see["Understanding result value averaging," Section 42.6.2](pt05ch42s04s02.md).

您还可以显示或隐藏视口中为您选择的每个节点显示的注释。探针注释显示节点的标签以及当前字段输出变量在该节点的结果。您可以自定义探针注释文本的字体和颜色。

**探测元件**

当您选择探测元素并将光标定位在当前视口中的元素上时，Abaqus/CAE 会突出显示该元素并在数据表中显示有关该元素的以下信息： 
- 元素的标签，
- 元素的连接性，以及
- 当前场输出变量（如果当前场输出变量是张量或向量，则包括分量值）。如果在当前视口中选择模型，则当前字段输出变量反映模型属性的值，例如所选元素的载荷。

仅当当前主变量是基于单元的变量（例如应力或应变）时，您才可以在单元处显示场输出。您可以选择计算场输出结果的输出位置：积分点、质心、单元节点或单元面。对于每个输出位置，Abaqus/CAE 逐个元素计算探测结果，不求平均值。

您还可以显示或隐藏视口中为您选择的每个元素显示的注释。探针注释显示元素的标签以及该元素当前字段输出变量的结果。您可以自定义探针注释文本的字体和颜色。

无论模型显示的渲染风格如何，都只能探测可见元素。如有必要，您可以创建显示组或使用视图剪切来显示其他不可用的内部元素。

当一个单元被裂缝切穿时，破裂的单元会分裂成两部分，每一部分由一个真实域和一个幻域组成。当您探测裂纹元素时，仅报告一部分元素的贡献。有关详细信息，请参阅["Visualization" in "Modeling discontinuities as an enriched feature using the extended finite element method," Section 10.7.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-aenrichment-vis)。
