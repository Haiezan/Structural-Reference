# 11.13.2 控制拉伸特征的方向







当您将拉伸特征添加到三维零件时，Abaqus/CAE 会根据您要创建的特征类型从草绘轮廓中选择默认拉伸方向。默认情况下，实体或壳特征向外拉伸，以便将材料添加到现有基础特征。相反，切割特征向内挤压，从而从现有基础特征中去除材料。

您可以按如下方式控制拉伸特征的方向：

**添加拉伸特征时选择方向**

当您完成草图以将拉伸特征添加到现有零件时，Abaqus/CAE 会在原始零件上显示新绘制的轮廓。绘制的轮廓包括一个指示挤出方向的箭头。 Abaqus/CAE 还显示 **编辑拉伸** 对话框。

您可以通过单击 **编辑拉伸** 对话框中的![](../graphics/ico_flipArrow.png)来控制拉伸方向。视口中的箭头改变方向以显示新的拉伸方向。

**编辑现有拉伸特征的方向**

当您选择要编辑的拉伸特征时，Abaqus/CAE 会在视口中突出显示所选特征，并出现 **编辑特征** 对话框。

您可以通过在 **编辑特征** 对话框中切换 **翻转拉伸方向** 来反转拉伸方向。 Abaqus/CAE 不显示指示拉伸方向的箭头；但是，您可以单击“**应用**”来查看您的更改。当方向可接受时，单击 **确定** 结束编辑过程。
