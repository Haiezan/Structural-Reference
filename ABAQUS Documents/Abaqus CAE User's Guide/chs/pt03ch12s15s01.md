# 12.15.4 指定材料方向或钢筋参考方向







您可以指定材料方向或钢筋参考方向。以下部分描述了如何分配这些方向：
-["Assigning a material orientation](pt03ch12s15s01.md#usi-prp-assign-matorient)”
-["Assigning a rebar reference orientation](pt03ch12s15s01.md#usi-prp-assign-rebarorient)”

### 分配材料方向

全局坐标系确定默认材料方向。但是，您可以通过选择基准坐标系或离散场或定义离散方向，将特定材料方向指定给壳、实体零件、区域、蒙皮或纵梁。对于 Abaqus/Standard 分析，您可以在用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中定义局部坐标系。基准坐标系可以是直角坐标系、柱坐标系或球坐标系；如果需要，您可以对坐标系应用额外的旋转。有关详细信息，请参阅["Orientations," Section 2.2.5 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-corientation)。

如果 Abaqus/CAE 尝试在系统的奇点处绘制材料方向的坐标系（即，用户选择的坐标系和几何体或元素的几何法线无法解析为 Abaqus/CAE 中显示目的的有效方向的点），则坐标系将被折叠绘制。

对于连续体壳单元，Abaqus/CAE 不会将显示的材料方向投影到中平面表面上。您可以执行数据检查并在可视化模块中查看输出数据库以验证材料方向。有关详细信息，请参阅["Performing a data check on a model," Section 19.7.3](pt03ch19s07hlb03.md)。

您分配给零件或区域的材料方向会自动分配给装配中该零件的所有实例。您可以使用查询工具集来验证是否已将正确的材料方向分配给选定区域。有关详细信息，请参阅["Using the Query toolset to obtain assignment information," Section 12.19](pt03ch12hla03.md)。

您可以编辑、抑制和恢复之前定义的材料方向，方法是在模型树中的材料方向上单击鼠标按钮 3，然后从显示的列表中选择抑制或恢复选项。有关详细信息，请参阅["Suppressing and resuming objects," Section 3.4.3](pt01ch03s04s03.md)。

**要为零件指定材料方向：**

1. 如果要指定材料方向的零件在当前视口中不可见，请在上下文栏中的 **零件** 列表中单击所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏中，选择****分配**![](../graphics/images/arrow.gif)**材料方向****。 **提示：**您还可以单击属性模块工具箱中的![](../graphics/ico_assignMaterialOrientation.png)工具。
3. 从视口中选择壳或实体区域，然后单击鼠标按钮 2 以指示您已完成选择。 （有关详细信息，请参阅[Chapter 6, "Selecting objects within the viewport](pt01ch06.md)。"） **提示：**您可以使用 **选择** 工具栏中的工具来限制可以在视口中选择的对象类型。有关详细信息，请参阅["Using the selection options," Section 6.3](pt01ch06s03.md)。如果选择 **蒙皮** 作为对象类型，则可以将材质方向分配给整个蒙皮或其一个或多个面。从提示区域中，选择 **（拾取整个蒙皮）** 或 **（拾取部分蒙皮）**，然后进行选择如果所选零件有多个蒙皮，则在进行选择时，Abaqus/CAE 将在提示区域中显示不明确的拾取选项。如果选择 **纵梁** 作为对象类型，则可以将材料方向分配给整个纵梁或其一个或多个边缘，选择 **（拾取整个纵梁）** 或 **（拾取部分纵梁）**，然后在视口中进行选择。当您进行选择时，Abaqus/CAE 将在提示区域中显示不明确的拾取选项，请执行以下操作： 1. 单击提示区域右侧的 **区域选择** 对话框，其中包含仅包含壳体或实体的零件组。 2. 选择感兴趣的零件组，然后单击 **继续**。 **注意：**默认选择方法基于您最近使用的选择方法。要恢复为其他方法，请单击提示区域右侧的“在视口中选择”或“设置”。
4. 如果要使用现有基准坐标系指定材料方向，请执行以下操作： 1. 选择基准坐标系。 - 在视口中选择坐标系。 - 按名称选择基准坐标系。在提示区域单击“**基准坐标系列表**”，从列表中选择一个名称，然后单击“**确定**”。 Abaqus/CAE 显示为该区域指定的结果材料方向，并出现 **编辑材料方向** 对话框。 2. 选择**轴 1**、**轴 2** 或 **轴 3** 以指定坐标系轴。 - 对于壳区域，在对话框的 **法线方向** 部分中指定近似于壳法线的坐标系轴。 - 对于实体​​区域，在对话框的 **附加旋转方向** 部分中指定发生可选附加旋转的坐标系轴。 3. 如果需要，您可以使用以下方法之一提供围绕壳法线（对于壳区域）或围绕基准坐标系的局部轴（对于实体区域）的材料方向的附加旋转： - 选择“**角度**”，然后输入附加旋转的度数值。 - 选择 **分布**，然后选择一个现有的标量离散场，该场定义在整个区域中空间变化的角度。或者，您可以单击![](../graphics/ico_discreteField.png)创建新的离散字段。 （有关详细信息，请参阅“[Chapter 63, "The Discrete Field toolset](pt06ch63.md)”。）Abaqus/CAE 应用附加旋转（如果有），并显示应用于该区域的结果材料方向。
5. 如果要使用全局坐标系（默认）或在离散方向、用户子例程或离散字段中指定材料方向，请从提示区域单击 **使用默认方向或其他方法**。您还可以选择现有基准坐标系或使用此过程创建新基准坐标系。 1. 在出现的 **编辑材料方向** 对话框中，单击 **定义** 字段右侧的箭头，然后选择以下选项之一： - 选择 **坐标系**，然后单击![](../graphics/ico_selectBlue.png)从视口或按名称选择坐标系。或者，您可以单击![](../graphics/ico_dtmCsys.png)创建新的基准坐标系。 - 选择“**全局**”以使用全局坐标系或将之前指定的材料方向重置回全局坐标系。 - 选择**离散**，然后单击![](../graphics/ico_edit.png)定义离散方向。在出现的 **编辑离散方向** 对话框中，使用["Using discrete orientations for material orientations and composite layup orientations," Section 12.16](pt03ch12hla01.md)中描述的过程定义法线轴和主轴。 - 选择**用户定义**以定义用户子程序[`ORIENT`](../sub/sub-link.md#sub-xsl-orient)中的方向。此选项仅对 Abaqus/Standard 分析有效。有关详细信息，请参阅以下部分： -["Specifying general job settings," Section 19.8.6](pt03ch19s08hlb06.md)-["ORIENT," Section 1.1.15 of the Abaqus User Subroutines Reference Guide](../sub/sub-link.md#sub-rtn-uorient)- 选择离散字段以定义空间变化的方向。或者，您可以单击 **定义** 字段右侧的![](../graphics/ico_discreteField.png)以创建新的离散字段。 （有关详细信息，请参阅“[Chapter 63, "The Discrete Field toolset](pt06ch63.md)”。） 2. 选择 **轴 1**、**轴 2** 或 **轴 3** 以指定坐标系轴。 - 对于外壳区域，在对话框的 **法线方向** 部分中指定近似于外壳法线的坐标系轴。 - 对于实体区域，在对话框的 **附加旋转方向** 部分中指定发生可选附加旋转的坐标系轴。 3. 如果如果需要，您可以使用以下方法之一提供围绕壳体法线（对于壳体区域）或围绕基准坐标系的局部轴（对于实体区域）的附加旋转： - 选择 **角度**，并输入附加旋转的度数值 - 选择 **分布**，然后选择定义在整个区域中空间变化的角度的现有标量离散字段。[Chapter 63, "The Discrete Field toolset](pt06ch63.md)，”了解更多信息。）Abaqus/CAE 应用附加旋转（如果有），并显示应用于该区域的结果材料方向。 4. 选择以下选项之一来指定连续体壳单元或实体单元的堆叠方向： - **单元等参方向 1** - **单元等参方向 2** - **单元等参方向 3（从下到上）** - **材料取向的法向方向（仅限连续体壳）**。堆叠方向是材料取向的3个方向。如果选定区域已网格化，Abaqus/CAE 将显示每个单元的材料堆叠方向。
6. 在对话框中单击“**确定**”。

### 指定钢筋参考方向

钢筋层的角度方向是相对于钢筋参考方向定义的。全局坐标系确定默认的钢筋参考方向。但是，您可以通过从视口中选择现有基准坐标系，然后在基准坐标系上选择近似于壳体法线方向的轴，将钢筋参考方向指定给壳体零件、区域、蒙皮或纵梁。基准坐标系可以是直角坐标系、柱坐标系或球坐标系；如果需要，您可以对坐标系应用额外的旋转。有关详细信息，请参阅["Orientations," Section 2.2.5 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-int-corientation)。

您指定给零件或区域的钢筋参考方向会自动指定给装配中该零件的所有实例。您可以使用查询工具集来验证是否已将正确的钢筋参考方向分配给选定区域。有关详细信息，请参阅["Using the Query toolset to obtain assignment information," Section 12.19](pt03ch12hla03.md)。

**要将钢筋参考方向分配给零件：**

1. 如果要为其指定钢筋参考方向的零件在当前视口中不可见，请在上下文栏中的 **零件** 列表中单击所需零件的名称。所选零件出现在当前视口中。
2. 从主菜单栏中，选择****分配**![](../graphics/images/arrow.gif)**钢筋参考方向****。 **提示：**您还可以单击属性模块工具箱中的![](../graphics/ico_assignRebarOrientation.png)工具。
3. 从视口中选择区域，然后单击鼠标按钮 2 以指示您已完成选择。 （有关详细信息，请参阅[Chapter 6, "Selecting objects within the viewport](pt01ch06.md)。"） **提示：**您可以使用 **选择** 工具栏中的工具来限制可以在视口中选择的对象类型。有关详细信息，请参见["Using the selection options," Section 6.3](pt01ch06s03.md)。如果选择 **蒙皮** 作为对象类型，则可以将钢筋参考方向分配给整个蒙皮或其一个或多个面。从提示区域中，选择 **（拾取整个蒙皮）** 或 **（拾取部分蒙皮）**，然后使如果所选零件有多个蒙皮，则在进行选择时，Abaqus/CAE 将在提示区域中显示不明确的拾取选项。如果选择 **纵梁** 作为对象类型，则可以将钢筋参考方向分配给整个纵梁或其一个或多个边缘。从提示区域中，选择 **（拾取整个纵梁）** 或 **（拾取部分纵梁）**，然后在视口中进行选择。当您进行选择时，Abaqus/CAE 将在提示区域中显示不明确的拾取选项，请执行以下操作： 1. 单击提示区域右侧的 **区域选择** 对话框，其中包含可用的集合列表 2. 选择感兴趣的零件组，然后单击 **继续** 注意：**默认选择方法基于您最近使用的选择方法。要恢复到其他方法，请单击提示区域右侧的“**在视口中选择**”或“**集**”。
4. 使用以下方法之一在选定区域中指定局部坐标系： - 在视口中选择坐标系。 - 按名称选择基准坐标系。在提示区域单击“**基准坐标系列表**”，从列表中选择一个名称，然后单击“**确定**”。 - 单击“**使用默认方向**”以使用全局坐标系。 Abaqus/CAE 显示为该区域指定的结果钢筋方向，并出现 **编辑钢筋方向** 对话框。
5. 选择**轴 1**、**轴 2** 或 **轴 3** 以指定坐标系轴。 - 对于壳区域，在对话框的 **法线方向** 部分中指定近似于壳法线的坐标系轴。 - 对于实体​​区域，在对话框的 **附加旋转方向** 部分中指定发生可选附加旋转的坐标系轴。
6. 如果需要，您可以提供围绕壳法线（对于壳区域）或围绕基准坐标系的局部轴（对于实体区域）的钢筋参考方向的附加旋转。选择 **角度**，然后输入附加旋转的度数值。 Abaqus/CAE 应用附加旋转（如果有），并显示应用于该区域的结果钢筋参考方向。
7. 在对话框中单击“**确定**”。
