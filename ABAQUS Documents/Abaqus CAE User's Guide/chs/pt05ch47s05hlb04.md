# 47.6.5 自定义 X-Y 曲线上使用的符号







### 47.6.5 自定义 X--Y 曲线上使用的符号

使用 **曲线选项** 对话框自定义用于表示 *X–Y* 曲线上数据点的符号的外观。例如，在[Figure 47--43](pt05ch47s05hlb04.md#viw-symbolopt)中，左侧图使用默认符号，而右侧图使用自定义符号。

**图 47-43** 具有不同数据点符号的 X-Y 图。

![](../graphics/viw-symbolopt.png)

您选择的符号将出现在曲线上和[legend](pt05ch56s01.md)中。仅当打开 **显示符号** 时，自定义选项才可用。

**要自定义 X-Y 曲线上使用的符号：**

1. 找到**属性**选项。 **属性**选项出现在**曲线选项**对话框的下半部分。
2. 在 **Curves** 字段中，选择您想要自定义其符号的[select one or more *X--Y* curves](pt05ch47s05s01.md)。 **注意：**要使 *X--Y* 曲线可供选择，您必须首先绘制它。您选择的自定义选项将应用于您选择的所有曲线；如果您想要改变多条曲线上的符号，则必须一次自定义一个符号。
3. 切换 **显示符号** 以显示或隐藏代表每个选定 *X--Y* 曲线数据点的符号。当**显示符号**打开时，将显示符号并启用符号属性。
4. 选择符号颜色： 1. 单击颜色样本![](../graphics/usi-colorsample.png)。 Abaqus/CAE 显示 **选择颜色** 对话框。 2. 使用“**选择颜色**”对话框中的方法之一选择新颜色。有关更多信息，请参阅["Customizing colors," Section 3.2.9](pt01ch03s02s09.md)。 3. 单击“**确定**”关闭“**选择颜色**”对话框。颜色样本更改为选定的颜色。在您单击“**XY 绘图选项**”对话框中的“**确定**”或“**应用**”之前，新颜色不会出现在视口中。
5. 选择符号： 1. 单击 **符号** 按钮以显示符号选择。 2. 从符号列表中，单击所需的符号。指定的符号出现在选定的 *X--Y* 曲线上。
6. 选择符号大小： 1. 单击 **大小** 按钮以显示符号大小选项。 2. 从符号大小选项中，单击所需的符号大小。指定的符号大小出现在选定的 *X--Y* 曲线上。
7. 选择符号的频率。在 **频率** 文本字段中输入一个整数，以选择符号沿曲线出现的频率。例如，如果输入频率为 2，Abaqus 将每隔一个数据点显示符号。不允许使用零值；要禁止显示符号，请关闭 **显示符号**。
8. 单击“**关闭**”关闭“**曲线选项**”对话框。
