# 10.9.6 将视口数据导出为 OBJ 格式文件







从主菜单栏中选择****文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**OBJ****，将当前视口中的模型导出为OBJ格式的文件。从可视化模块导出数据时，Abaqus/CAE 仅将网格数据导出到 OBJ 格式文件。如果从任何其他模块导出数据，并且网格显示在当前视口中，则 Abaqus/CAE 会导出网格数据；否则，Abaqus/CAE 会导出几何数据。

从可视化模块导出对当前步骤和帧敏感。如果在视口中显示未变形图，Abaqus/CAE 会将未变形数据导出到 OBJ 格式文件；如果显示变形图，Abaqus/CAE 会将当前步骤和帧的变形数据导出到 OBJ 格式文件。

Abaqus/CAE 在导出过程中对几何和网格数据执行以下转换：
- 模型数据库中的实体元素转换为 OBJ 格式文件中的壳元素。
- 模型数据库中的四边形元素转换为 OBJ 格式文件中的三角形元素。
- 模型数据库中的二阶元素转换为 OBJ 格式文件中的一阶元素。如果您的模型使用大量二阶单元进行网格划分，则导出过程可能会创建一个大型 OBJ 文件，因为 Abaqus/CAE 将二阶单元拆分为生成的一阶单元。

您导出的几何体或网格数据的质量以及生成的文件大小均取决于曲线细化设置。有关更改曲线细化质量的更多信息，请参阅["Controlling curve refinement," Section 76.4](pt07ch76hla03.md)。

**要将视口数据导出到 OBJ 格式文件：**

1. 如果要从可视化模块导出数据，请选择要导出数据的绘图状态。要从变形绘图状态导出数据，请选择要导出数据的步骤和框架。有关选择当前步骤和帧的更多信息，请参阅["Selecting the results step and frame," Section 42.3](pt05ch42s01.md)。
2. 从主菜单栏中，选择****文件**![](../graphics/images/arrow.gif)**导出**![](../graphics/images/arrow.gif)**OBJ****。将出现 **导出到 OBJ 文件** 对话框。
3. 从对话框底部的“**文件过滤器**”菜单中，选择“**OBJ (*.obj)**”。 Abaqus/CAE 列出了所选目录中文件扩展名为`.obj`的所有文件。
4. 选择要将几何图形和网格数据导出到的文件，或在 **文件名** 文本字段中键入新文件的名称。
5. 单击“**确定**”导出数据并关闭“**导出到 OBJ 文件**”对话框。
