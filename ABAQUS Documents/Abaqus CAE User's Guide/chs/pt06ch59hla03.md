# 59.5 基于边缘创建附着点图案







您可以沿着单条边或连接的边以简单图案放置附着点。然后，您可以将这些点放置在相邻面上或沿指定方向的行图案中。如果需要，您可以将指定的点投影到选定的面上。您可以使用连接点来定义基于点的紧固件的位置。有关详细信息，请参阅["About fasteners," Section 29.1](pt04ch29s01.md)。

**根据边缘创建附着点图案：**

1. 从主菜单栏中，选择****工具**![](../graphics/images/arrow.gif)**附件**![](../graphics/images/arrow.gif)**点与边的偏移****。 **提示：**您还可以通过使用![](../graphics/ico_attPtOffEdge.png)工具选择边和偏移来创建附着点，该工具位于属性模块或相互作用模块工具箱中的附着工具中。有关工具箱中附件工具的更多信息，请参阅["Understanding attachment points and lines," Section 59.1](pt06ch59s01.md)。
2. 从当前视口中，选择要分配附着点的边。您可以选择多条边，只要边相连并且不分支即可。您可以单击提示区域右侧的“**集合**”来使用之前定义的集合。如果边选择形成闭环，则从视口中选择点生成的起始顶点。出现红色箭头指示从起点开始的点生成方向。
3. 在提示区域，单击“**翻转**”以反转点生成的方向（如有必要），然后单击“**是**”。 Abaqus/CAE 显示 **创建从边偏移的附着点** 对话框。
4. 在对话框中，指定沿边的点。 - 选择 **按数字** 以定义沿边缘选择的附着点数量。 Abaqus/CAE 自动沿边缘间隔点。 1. 输入沿边缘的点数。 2. 输入第一个附着点相对于起点的所需偏移量。 3. 输入最后一个连接点相对于终点的所需偏移量。 - 选择**按间距**来定义附着点之间的距离。 1. 输入点之间的间距。 2. 输入第一个附着点相对于起点的所需偏移量。 3. 指定沿边缘的点数，或选择 **自动调整** 让 Abaqus/CAE 确定附着点的数量。如果指定点数，Abaqus/CAE 将仅创建起点和终点之间的点；任何延伸超过终点的点都不会被创建。
5. 如果需要，可以通过指定以下内容来定义连接点的行图案： - 输入图案中的行数。默认值 1 创建单行附着点。 - 输入间距值以定义图案行之间的距离。 - 输入图案第一行距边缘的偏移距离。默认值 0 将在边缘创建图案的第一行。 - 选择 **正交于面** 作为偏移方法，然后单击![](../graphics/ico_selectBlue.png)选择要放置阵列的相邻面。 - 选择**沿方向**作为偏移方式，然后单击![](../graphics/ico_selectBlue.png)指定方向矢量的起点和终点。
6. 要将指定点投影到选定面上，请显示 **投影** 选项卡页并执行以下操作： 1. 切换到 **投影到面上**。 2. 单击![](../graphics/ico_selectBlue.png)，然后选择点将投影到的面。这些面可以是平面的或非平面的。 3. 选择投影点的方法。 - 选择 **Proximity** 以允许 Abaqus/CAE 绘制从每个点到所选面上最近点的矢量。 - 选择 **方向** 定义沿其投影点的矢量。单击![](../graphics/ico_selectBlue.png)选择矢量的 **起点** 和 **终点**。有关更多信息，请参阅["Understanding the projection methods," Section 59.2](pt06ch59s02.md)。
7. 默认情况下，Abaqus/CAE 创建一个包含附着点的集。如果需要，您可以修改集名称。如果您不想创建包含附着点的集，请关闭**使用名称创建集**。
8. 单击“**确定**”创建连接点。 Abaqus/CAE 显示连接点。有关编辑附着点的信息，请参阅["Editing attachment points created by a pattern based on edges," Section 59.7.2](pt06ch59s03hlb02.md)。
