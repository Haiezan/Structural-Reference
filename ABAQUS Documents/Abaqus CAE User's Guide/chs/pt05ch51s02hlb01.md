# 51.2.1 探测选项概述





您可以使用查询工具集中的探测选项来指定在探测模型或 X–Y 绘图时想要获取的值。点击可视化模块工具箱中的 ![](../graphics/ico_probe.png) 访问 **探针值（Probe Values）** 对话框。使用此对话框中的选项来控制当前视口中的探测。

**对于来自当前模型数据库的模型数据**

使用 **场输出（Field Output）** 对话框来选择步骤和场输出变量。对于模型数据，您选择的场输出变量实际上是模型中属性的值，如载荷或预定义场。

**对于模型绘图**

使用 **场输出（Field Output）** 对话框来选择步骤、帧和场输出变量。使用 **结果选项（Result Options）** 对话框来选择控制 Abaqus/CAE 显示的探测数据和结果的计算方法。

**对于模型和 X–Y 绘图**

使用 **探针值（Probe Values）** 选项来选择在探测时想要获取的信息类型。
