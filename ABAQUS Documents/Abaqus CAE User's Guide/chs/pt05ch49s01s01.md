# 49.1.1 时间历史动画





在时间历史动画中，Abaqus/CAE 通过为活动步骤和帧在输出数据库中找到的每个指定帧重绘变形形状、云图、符号或材料取向绘图来创建序列中的图像。时间历史动画中的每个图像显示实际分析结果。变形比例因子在时间历史动画的每个图像中保持不变。结果帧可以按顺序绘制，也可以根据时间间隔从活动帧中选择。

如果您正在为云图制作动画，场输出值的显示取决于您选择的选项。默认行为是图例中的值基于第一个和最后一个动画帧的最小值和最大值固定。要了解如何设置云图限制，请参阅 ["设置云图限制，" 第 44.5.7 节"](pt05ch44s02hlb07.md)。
