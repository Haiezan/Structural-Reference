# 11.19.2 选择新零件的建模空间







使用“创建零件”对话框顶部的“建模空间”单选按钮来选择要创建的零件的建模空间。 Abaqus/CAE 在建模过程中携带零件的建模空间；例如，建模空间决定部件模块中哪些工具可用以及网格模块中哪些单元可用。创建零件后，您可以通过在模型树中的零件上单击鼠标按钮 3 并从出现的菜单中选择 **编辑** 来更改零件的建模空间。

建模空间是指零件可以居住的空间，而不是零件本身。因此，您可以在三维建模空间中创建零件，但使用拓扑二维壳或线特征构造它。在 Abaqus/CFD 模型中只能创建三维零件。在 Abaqus/Standard 或 Abaqus/Explicit 模型中，新零件的建模空间可以设置为以下任意一项：

**三维**

Abaqus/CAE 将零件定位在三维空间中。

**二维平面**

Abaqus/CAE 将零件定位在平面二维空间中。

**轴对称**

Abaqus/CAE 将零件定位在轴对称二维空间中。如果您创建轴对称可变形零件，Abaqus/CAE 允许您在模型中包含扭转自由度。

**选择新零件的建模空间：**

1. 从 **创建零件** 对话框的顶部，选择所需的 **建模空间** 单选按钮。
2. 选择完选项后，单击“**继续**”关闭“**创建零件**”对话框。草绘器启动，您可以绘制新零件基本特征的轮廓。
