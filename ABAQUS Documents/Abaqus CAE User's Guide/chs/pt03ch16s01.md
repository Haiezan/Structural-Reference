# 16.1 理解Load模块的作用







Abaqus/CAE 中的规定条件是步骤相关的对象，这意味着您必须指定它们处于活动状态的分析步骤。您可以使用载荷、边界条件和预定义字段管理器来查看和操作规定条件的逐步历史记录。您还可以使用上下文栏中的**步骤**列表来指定默认情况下新载荷、边界条件和预定义字段处于活动状态的步骤。

您可以使用“加载”模块中的“幅度”工具集来指定可应用于规定条件的复杂时间或频率依赖性。载荷模块中的设置和曲面工具集允许您定义和命名要应用规定条件的模型区域。分析字段工具集和离散字段工具集允许您创建可用于为选定的规定条件定义空间变化参数的字段。

载荷工况是用于定义特定载荷条件的载荷和边界条件集。您可以在静态扰动和稳态动态、直接步骤中创建载荷工况。有关载荷工况的信息，请参阅[Chapter 34, "Load cases](pt04ch34.md)。”![](../graphics/images/black4rule.gif)有关相关主题的信息，请单击以下任意项目：-["Prescribed conditions: overview," Section 34.1.1 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-prc-pprescribedover)-["Multiple load case analysis," Section 6.1.4 of the Abaqus Analysis User's Guide](../usb/usb-link.md#usb-anl-amultiloadcase)-[Chapter 57, "The Amplitude toolset](pt06ch57.md)”
