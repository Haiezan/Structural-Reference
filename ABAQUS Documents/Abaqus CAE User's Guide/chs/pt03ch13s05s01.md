# 13.5.1 装配模块中的定位工具







每个零件都存在于部件模块中自己的坐标系中，并且模型实例在其自己的坐标系中创建。您可以使用装配模块在全局坐标系中相对于彼此定位和定向这些零件和模型的实例。 Abaqus/CAE 提供以下工具用于定位零件和模型实例：

**自动偏移**

当您在装配模块中创建第一个零件或模型实例时，Abaqus/CAE 会显示一个三重轴，指示全局坐标系的原点和方向。 Abaqus/CAE 定位第一个实例，使零件或模型的原点与全局坐标系的原点对齐，并且轴对齐。如果您创建其他实例，Abaqus/CAE 会继续定位新实例，使其坐标系与全局坐标系对齐。由于这通常会导致新实例与现有实例重叠，因此 Abaqus/CAE 允许您在创建实例之前应用偏移。对于三维和二维实例，偏移量沿 *X* 轴应用；对于轴对称实例，偏移量沿 *Y* 轴应用。

**基本定位工具**

Abaqus/CAE 提供了以下用于定位零件和模型实例的基本方法：
- 您可以通过指定平移矢量的起点和终点的坐标来沿矢量平移选定的实例。您可以使用以下方法来确定所选实例移动的距离： - 所选实例沿着平移向量从起点移动到终点。 - 选定的实例沿平移向量从起点向终点移动，并继续移动，直到选定的面或边与从固定实例中选定的面或边达到指定距离。有关详细信息，请参阅["Positioning a part or model instance using the **Translate To** tool," Section 13.5.4](pt03ch13s05s04.md)。
- 您可以绕轴旋转选定的实例。您指定旋转轴的起点和终点的 *X*-、*Y*- 和 *Z*- 坐标以及旋转角度。

**位置约束工具**

位置约束定义两个实例之间的关系。与简单的平移或旋转不同，您不直接指定位置。位置约束定义了装配体中的零件或模型实例必须始终满足的一组规则；例如，一个面必须与另一个面平行。

装配模块中定义的位置约束仅对实例的初始位置创建约束，而相互作用模块中定义的约束定义分析自由度的约束。在装配模块中，约束被存储为装配的特征。如果修改零件或移动零件或模型实例，Abaqus/CAE 会在重新生成装配时尝试应用所有现有位置约束。每个位置约束均在["How the position constraint methods differ," Section 13.5.2](pt03ch13s05s02.md)中描述。

创建最终装配是一个创建实例、应用位置约束以及应用平移和旋转的迭代过程。每次重新定位后，Abaqus/CAE 都会显示一个临时图像，指示操作的结果。您可以通过单击提示区域中的**上一步**按钮![](../graphics/afxI_backup.png)接受新位置、取消操作或后退重新定位过程。

您可以使用查询工具集获取顶点的坐标并测量选定顶点之间的距离。这可以帮助您确定需要平移实例的矢量或需要旋转实例的角度。["Using the Query toolset to query the assembly," Section 13.12](pt03ch13hla01.md)，包含有关如何获取有关程序集信息的详细说明。
