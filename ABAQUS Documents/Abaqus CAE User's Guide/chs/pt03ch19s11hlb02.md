# 19.11.2 对共同执行执行数据检查







数据检查通过分析输入文件处理器运行模型的输入文件，以确保模型一致并且已设置所有必需的模型选项。对协同执行执行数据检查将验证为协同执行创建的每个作业中的模型信息。

要对共同执行执行数据检查，请从主菜单栏中选择****共同执行**![](../graphics/images/arrow.gif)**数据检查**![](../graphics/images/arrow.gif)***共同执行名称*****。您还可以通过在**协同执行管理器**中选择协同执行名称并单击**数据检查**来执行数据检查。

共同执行中涉及的作业出现在模型树中的 **作业** 容器中，位于 **共同执行** 容器中的共同执行下。您可以通过在模型树中的作业上单击鼠标按钮 3 并选择 **监视器** 以显示作业监视器对话框来查看数据检查的结果。有关详细信息，请参阅["Monitoring the progress of an analysis job," Section 19.2.6](pt03ch19s02hlb01.md)。![](../graphics/images/black4rule.gif)有关相关主题的信息，请单击以下任意项目：-["Understanding co-executions," Section 19.4](pt03ch19s04.md)-["Creating, editing, and manipulating co-executions," Section 19.11](pt03ch19s11.md)
