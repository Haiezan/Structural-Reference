# 13.10.3 使用模型树切换零件实例或模型实例的上下文

您可以在装配模块或网格模块中使用模型树，切换零件实例、模型实例及其子零件实例的上下文。

在模型树中，选择要切换上下文的实例。如果选择多个实例，上下文将切换到您首先选择的实例对应的零件或模型。单击鼠标按钮 3，然后选择 **Switch to part/model context**。Abaqus/CAE 将根据您的选择按如下方式切换上下文：

- 在装配模块中，如果选择了零件实例或子零件实例，上下文将切换到零件模块，并在视口中显示创建该实例所基于的原始零件。
- 在网格模块中，如果选择了零件实例或子零件实例，上下文将切换为在视口中显示创建该实例所基于的原始零件。
- 如果选择了模型实例，上下文将切换为在视口中显示创建该模型实例所基于的原始模型。
