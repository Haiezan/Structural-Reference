# 11.19.4 选择新零件的基本特征







使用“**创建零件**”对话框底部的“**基本特征**”字段中的单选按钮和列表来选择要创建的零件的基本特征。这些选择取决于零件的建模空间和零件的类型；例如，轴对称可变形体可以仅具有平面壳、平面线或点基特征。有关可以创建的不同形状和类型的基础特征的详细信息，请参见["The base feature," Section 11.3.2](pt03ch11s03s02.md)。

您对基本特征类型的选择很重要，因为创建零件后无法更改类型。您可以修改基本特征，但您应该注意，随后添加到零件的任何特征都将链接到基本特征。因此，如果修改基本特征，这些从属（或子）特征可能会移动或无法重新生成。

**选择基本功能：**

1. 从 **创建零件** 对话框的底部，选择所需的基本特征形状（**实体**、**壳**、**线** 或 **点**）。可用的选择取决于建模空间和您正在创建的零件的类型。
2. 如果要创建三维可变形或离散刚性零件，还必须选择其**类型**（**拉伸**、**旋转**、**扫描**、**平面**或**坐标**）。
3. 选择完选项后，单击“**继续**”关闭“**创建零件**”对话框。草绘器启动，您可以绘制新零件基本特征的轮廓。
