# 19.6.10 远程提交重启动作业

如果将作业提交到远程计算机进行分析，默认情况下，作业完成后 Abaqus 会将重启动分析所需的文件复制回本地目录。更多信息请参见 ["远程提交作业," 第 19.2.7 节](pt03ch19s02s06.md)。如果远程作业提前终止，您可能必须手动将这些文件复制回本地目录。

重启动分析所需的大多数文件都是与平台相关的二进制文件。因此，如果在 UNIX 计算机上开始分析，则不能在 Windows 计算机上继续重启动分析，反之亦然。零件（`.prt`）文件包含 ASCII 文本，可以跨平台复制。

不同二进制格式的平台之间缺乏可移植性，并不排除在不同于原始分析运行机器的另一台机器上重启动分析。但是，两台机器必须二进制兼容，并且建议原始分析和重启动分析都在同一平台上执行。Abaqus 不测试重启动文件的跨平台兼容性。
