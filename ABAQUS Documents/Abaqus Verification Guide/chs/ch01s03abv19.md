# 1.3.16 悬臂夹芯梁：剪切柔性壳

**产品：** Abaqus/Standard

### 测试单元

S4    S4R    S8R    S4T    S4RT    S8RT

### 问题描述

![](../graphics/versandwichbeam-model.png)

**材料：**

面板采用线弹性材料，Young 模量 = 1.0  107，Poisson 比 = 0。芯材的横向剪切模量为 1.0  104，平面内所有其他属性使用 LAMINA 定义设置为可忽略值。

**边界条件：**

一端所有节点固支。

**载荷：**

![](../graphics/ver_eqn00160.gif) 750.0，一致分配到自由端节点。

S4、S4R 和 S8R 单元的壳截面使用 Gauss 积分。

S4T、S4RT 和 S8RT 单元的壳截面使用 Simpson 积分。

### 参考解

自由端位移（Plantema, *Sandwich Construction*, John Wiley and Sons, Inc., 1966）：![](../graphics/ver_eqn00172.gif)= 5.5684。

在此处强制防止翘曲的情况下，固支端顶部的最大弯曲应力为：![](../graphics/ver_eqn00173.gif) = 3.7275  105。
