# 1.3.20 轴对称膜单元

**产品：** Abaqus/Standard

### 测试单元

MAX1    MAX2    MGAX1    MGAX2

### 问题描述

![](../graphics/veraximemb-model-nls.png)

**模型：**

膜厚度为 0.05。

**材料：**

对于不使用方向的测试：线弹性，Young 模量 = 105，Poisson 比 = 0.3，热膨胀系数 = 107。

对于使用方向的测试：线弹性，工程常数为 ![](../graphics/ver_eqn00189.gif) 102，![](../graphics/ver_eqn00190.gif) 108，![](../graphics/ver_eqn00191.gif) 102，![](../graphics/ver_eqn00192.gif) 0，以及 ![](../graphics/ver_eqn00193.gif) 102。

**边界条件：**

底部节点的自由度 2 固定。此外，对于支持扭转的单元，底部节点的自由度 5 也固定。

**初始条件：**

对于不使用方向的测试，对所有单元施加 ![](../graphics/ver_eqn00194.gif) 0.001 和 ![](../graphics/ver_eqn00195.gif) 0.001 的初始应力场。所有节点的初始温度设为 0。

### 历史定义 I（适用于所有单元类型）

**步骤 1（几何非线性）：**

载荷：约束所有节点的所有自由度。建议使用该步骤施加初始应力。在后续步骤中施加所有必要边界条件。

**步骤 2（扰动）：**

载荷：在顶部节点施加方向 2 上大小为 314 的集中力。

解析解：顶部节点处 ![](../graphics/ver_eqn00196.gif) = 0.04998。

**步骤 3（扰动）：**

载荷：大小为 500 的内压。

解析解：环向应力 = 10000。

**步骤 4（扰动）：**

载荷：所有节点温度升高到 5000。

解析解：![](../graphics/ver_eqn00197.gif) 0.0005。

### 历史定义 II（适用于 MGAX1 和 MGAX2 单元类型）

**步骤 1（几何非线性）：**

载荷：约束所有节点的所有自由度。建议使用该步骤施加初始应力。在后续步骤中施加所有必要边界条件。

**步骤 2（扰动）：**

载荷：在顶部节点施加自由度 5 上大小为 200 的集中力矩。

解析解：剪应力 = 636.22。

### 历史定义 III（适用于使用局部坐标系的 MGAX1 和 MGAX2 单元类型）

**步骤 1（几何非线性）：**

载荷：在顶部节点施加大小为 2 的集中载荷。

**步骤 2（几何非线性）：**

载荷：在顶部节点施加大小为 2 的集中力矩。

### 结果与讨论

**历史定义 I：**

所有单元均给出精确解。

**历史定义 II：**

与解析解的偏差约为 0.06%。

**历史定义 III：**

结果与使用 CGAX4R（支持扭转的轴对称连续体单元）单元的类似细化模型结果比较。由于连续体模型中的厚度方向应变很小，膜模型的截面 Poisson 比设为 0。结果吻合良好。
