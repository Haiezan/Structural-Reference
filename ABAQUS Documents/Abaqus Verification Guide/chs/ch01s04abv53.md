# 1.4.3 Pipe 应力/位移单元

**产品：**Abaqus/Standard  Abaqus/Explicit

### I. 分布载荷

### 问题描述

内压施加在有效直径 3.6 上，外压施加在有效直径 4.0 上。同时测试有效轴向力输出变量 ESF1。

**模型：**

| 长度 | 15.0 |
| --- | --- |
| Pipe 截面数据 | *r* = 2.0, *t* = 0.2 |
| 离心旋转轴 | (0, 0, 1)，通过 (7.5, 0, 0) |
| 重力载荷向量 | (0, 1, 0) |

**材料：**

| Young 模量 | 3 106 |
| --- | --- |
| Poisson 比 | 0.3 |
| 密度 | 0.4188 |

**初始条件：**

| 初始温度 | ALL, 10.0 |
| --- | --- |
| 初始速度 | ALL, 1, 10.0 |
| (Coriolis 载荷) | ALL, 2, 5.0 |
| 静水压力基准值 | 100.0 |
| 静水压力标高 | 0.0 |

### 结果与讨论

计算得到的反力与施加的载荷一致。
