# 12.1 Abaqus 接触功能概述

![](../images/blu4rule.gif)

在 Abaqus/Standard 中，接触仿真既可以基于表面，也可以基于接触单元。在 Abaqus/Explicit 中，接触仿真仅基于表面。本指南讨论基于表面的接触。

基于表面的接触可以使用通用（“自动”）接触算法，也可以使用接触对算法。通用接触算法允许高度自动化的接触定义，其中接触基于自动生成的、包含全部可能接触区域的表面定义。相反，接触对算法要求显式地配对可能发生接触的表面。两种算法都需要指定表面之间的接触属性，例如摩擦。

本指南中的接触讨论涵盖 Abaqus/Standard 中的接触对方法和通用接触，以及 Abaqus/Explicit 中的通用接触。
